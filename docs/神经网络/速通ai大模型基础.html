什么是 AI？

我们认为，学习的每个方面或智能的任何其他特性原则上都可以被如此精确地描述，以至于可以制造一台机器来模拟它。我们将尝试找出如何使机器使用语言、形成抽象和概念、解决现在仅保留给人类的问题，并提高自己。我们认为，如果一个精心挑选的科学家团队在夏天一起工作，那么这一个或多个问题上都可以取得重大进展。
——《达特茅斯人工智能夏季研究项目提案》（McCarthy 等，1955 年）

什么是机器学习？

不依赖显示编程，让计算机通过算法自行学习并作出预测或决策。

什么是监督学习？

学习输入、输出的映射关系，能够准确给出预测。又分为：

  1. 分类：例如给一张图片，检测是猫还是狗。
  2. 回归：例如给一个房子，预测房价。

什么是无监督学习？

最重要的聚类。

什么是强化学习 ？

例如让模型下围棋，优化策略。

什么是深度学习？

深度学习主要在于模拟人的神经网络，有多层处理，最后给出一个答案。
深度学习可以用于上述三种任何一种学习。

什么是 transformer？


RNN 的弊端：

  1. 每一个输出决定下一个输出，不能进行并行运算。
  2. 不善于处理长序列。词语之间的距离越远，词语之间的关联越弱，难以捕捉长文本的语义关系。

RNN 的改良版本是 LSTM。

什么是 LSTM？

但是也没处理并行计算的问题。

什么是 transformer？


Transformer 会注意每一个词，并且给予每个词不一样的权重。


Transformer 的另一个创新就是位置编码。

Transformer 可以同时处理多个位置的信息，因为引入了一个重要的位置向量，因此可以进行并行运算，大大提升了速度。

详解 transformer

假如执行翻译认为，从英语翻译为法语。
当输入一句话之后，原始文本会被 token 化。


将 token 转成 ID，再转入嵌入层，将每个 token 转化成向量。

为什么又要用一串数字表示？因为能够包含多种语义信息。

向量有助于模型利用数学进行计算。

传入编码器

因为有自注意力机制，来计算不同词之间的相关性。

编码器有多个自注意机制，用来关注不同的方面。

之后，多个编码器同时计算。

传入解码器

解码器首先会接受一个值，表示开头。
解码器只会关注某个词与他前面的其他词，而不关注后面的其他词。确保生成文本时，遵循正确的时间顺序。被称为带掩码的自注意机制。

最后，线性层和 softmax 把解码器输出的表示转换为词汇表的概率分布。

但是有时候模型只关注概率而不关注逻辑，这种现象被称为幻觉 。
