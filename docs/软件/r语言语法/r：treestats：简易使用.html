什么是 treestats 包？

这是一个可以用于评价系统发育树的 各项统计指标 的 R 包，包括树的稳健性、分支长度等。

输入文件

需要输入一个树文件，推荐使用 nwk 格式。
这里我给一个示例数据： 生命之树.txt

代码

# 安装并加载必要的包（如果没有安装）
if (!require("treestats")) install.packages("treestats")
if (!require("RSpectra")) install.packages("RSpectra")
if (!require("ape")) install.packages("ape")
if (!require("treestats")) install.packages("treestats")
# 加载包
library(treestats)
library(ape)
library(RSpectra)
library(treestats)
# 读取树文件
phylo_tree <- read.tree("生命之树.txt")

# 打印树的概述
print(phylo_tree)

# 计算所有统计量并打印结果
results <- calc_all_stats(phylo_tree)
print(results)


# 假设你只对某些物种或支系感兴趣，可以列出不需要的叶节点
tips_to_remove <- c("Tip1", "Tip2", "Tip3")  # 替换为你想删除的叶节点名称
# 使用 drop.tip 函数来剪枝
sub_tree <- drop.tip(phylo_tree, tips_to_remove)
# 打印子树概述
print(sub_tree)
# 计算子树的统计量
sub_tree_results <- calc_all_stats(sub_tree)
print(sub_tree_results)


# 假设你知道感兴趣支系的共同祖先节点ID
ancestor_node <- 50  # 这是假设的节点编号，替换为你感兴趣的支系节点ID
# 使用 extract.clade 提取从该节点开始的子树
sub_tree <- extract.clade(phylo_tree, ancestor_node)
# 打印子树概述
print(sub_tree)
# 计算子树的统计量
sub_tree_results <- calc_all_stats(sub_tree)
print(sub_tree_results)

解读

  • area_per_pair：两个叶子节点之间的边数总和。衡量树的拓扑复杂度。
  • average_leaf_depth：平均叶子深度，指树中从根到叶子节点的平均路径长度。用于衡量树的平衡性。
  • avg_ladder：平均阶梯数，表示在树中存在的“阶梯”状结构的平均长度。较高的值表示树的形态更不平衡。
  • avg_vert_depth：平均顶点深度，表示从叶子节点到树根的平均路径长度。
  • b1：B1 平衡指标，衡量树的平衡性，较高的值表示树不平衡。
  • b2：B2 平衡指标，基于信息熵来衡量树的平衡性，数值越高树越平衡。
  • beta：Aldous’ beta 统计量，用于拟合节点的子代分裂情况，负值表示树的形态更不平衡。
  • blum：Blum 指数，衡量树的不平衡性，较高的值表示树更加不平衡。
  • cherries：樱桃对数，指树中相连的叶子节点对数。
  • colless：Colless 指数，衡量树的不平衡性，数值越高表示树越不平衡。
  • colless_corr：修正的 Colless 指数，进行树大小修正后的 Colless 指数。
  • colless_quad：Colless 指数的平方版本，提供更大的差异。
  • crown_age：树的冠年龄，表示树的最早分支到现在的时间长度。
  • diameter：树的直径，树中最长路径的长度。
  • double_cherries：双樱桃对，指一对樱桃节点同时连接到同一个节点。
  • eigen_centrality：特征向量中心性，衡量节点在整个树中的影响力。
  • eigen_centralityW：加权特征向量中心性，使用加权分支长度计算。
  • ew_colless：平衡的 Colless 指数，考虑到不同节点的大小。
  • four_prong：四分叉节点的数量，表示树中有四个叶子节点连接到同一个父节点的数量。
  • gamma：Gamma 统计量，用于衡量内部节点相对于树根的位置（NA表示无法计算）。
  • i_stat：I 统计量，衡量节点连接子代的大小不平衡性。
  • il_number：IL 数量，树中只有一个子叶节点的内部节点数。
  • imbalance_steps：不平衡步数，指将树变为完全不平衡树所需的步骤数。
  • j_one：J^1 指数，基于 Shannon 信息熵计算的树平衡性指数。
  • j_stat：J 统计量，用于衡量树的分化程度。
  • laplace_spectrum_a：拉普拉斯光谱的不对称性，衡量树的形状不对称性。
  • laplace_spectrum_e：拉普拉斯光谱的最大特征值。
  • laplace_spectrum_g：拉普拉斯光谱的特征间隙，表示特征值之间的差异。
  • laplace_spectrum_p：拉普拉斯光谱的峰值高度。
  • max_adj：最大邻接矩阵特征值。
  • max_betweenness：最大中介中心性，衡量节点在最短路径中的重要性。
  • max_closeness：最大接近中心性，表示节点与其他节点的距离接近程度。
  • max_closenessW：加权接近中心性，考虑分支长度。
  • max_del_width：最大宽度差，衡量树的不平衡性。
  • max_depth：最大深度，树中从根到叶子节点的最长路径。
  • max_ladder：最长的阶梯状结构。
  • max_laplace：最大拉普拉斯矩阵特征值。
  • max_width：树的最大宽度，表示在特定深度的最大节点数。
  • mean_branch_length：平均分支长度，表示树中分支的平均长度。
  • mean_branch_length_ext：平均外部分支长度，指连接叶子节点的分支的平均长度。
  • mean_branch_length_int：平均内部分支长度，指树内部节点之间的分支长度。
  • min_adj：最小邻接矩阵特征值。
  • min_laplace：最小拉普拉斯矩阵特征值。
  • mntd：最近种系距离，指每个叶子节点到最近的叶子节点的距离。
  • mpd：平均成对距离，所有叶子节点之间的平均距离。
  • mw_over_md：最大宽度与最大深度的比值。
  • nltt_base：标准化谱系通过时间的统计量。
  • number_of_lineages：谱系数量，表示树中的谱系数量。
  • phylogenetic_div：系统发育多样性，基于树的分支长度来衡量物种多样性。
  • pigot_rho：Pigot的Rho统计量，表示树在不同时间段内的分化速率变化。
  • pitchforks：三分支叉，表示树中有三个叶子节点连接到同一个父节点的数量。
  • psv：系统发育物种变异性，用于衡量物种间的系统发育相关性。
  • rogers：Rogers不平衡指数，基于内部节点的平衡性。
  • root_imbalance：树根节点的不平衡性。
  • rquartet：随机四分支统计量，表示树中随机四个节点之间的拓扑结构。
  • sackin：Sackin指数，衡量树的不平衡性。
  • stairs：阶梯结构的数量，衡量树中的阶梯状模式。
  • stairs2：另一种阶梯统计量。
  • symmetry_nodes：对称节点数，衡量树中对称节点的数量。
  • tot_coph：总共系距离，表示树中所有叶子节点之间的成对距离总和。
  • tot_internal_path：内部路径长度总和。
  • tot_path：路径总长度，表示树中所有路径的总长度。
  • tree_height：树的高度，表示从根到叶子的最大距离。
  • treeness：树的树状结构指数，表示树的分化程度。
  • var_branch_length：分支长度的方差。
  • var_branch_length_ext：外部分支长度的方差。
  • var_branch_length_int：内部分支长度的方差。
  • var_depth：树的深度方差。
  • vpd：成对距离的方差。
  • wiener：Wiener指数，衡量树的传递性，即所有节点之间距离的总和。

重要

人类 Y染色体 或 线粒体DNA（mtDNA）构建的系统发育树时，有几个特定的统计量和指标可以帮助我们更好地理解这些树的拓扑结构、进化过程以及遗传多样性。这些统计量能反映树的平衡性、分化速率以及不同族群的遗传结构。以下是特别需要关注的几个指标及其意义：

1. Sackin 指数 和 Colless 指数 （不平衡性指数）

  • 意义：这些指数衡量系统发育树的不平衡性，即不同分支在树上的分化程度是否一致。Y染色体和线粒体DNA由于单亲遗传，往往呈现明显的不平衡现象。高Sackin或Colless值表示某些谱系可能经历了快速扩张或特定事件（如人口迅速扩张等）。
  • 重要性：对于研究人类遗传谱系中的 瓶颈效应、快速扩张事件 或 祖先遗传谱系的集中性，这些指数非常重要。

2. Crown Age（冠年龄）

  • 意义：表示树的最早分支到现在的时间长度。在Y染色体和线粒体DNA的研究中，冠年龄能告诉我们最早的共同祖先出现的时间。
  • 重要性：帮助确定 最近的共同祖先（MRCA） 的时间，尤其是在推测人类起源、迁徙路线以及不同族群的分化时间时具有重要作用。

3. 树的高度（Tree Height）

  • 意义：树高度反映从根部（即共同祖先）到最远叶子节点的距离。对于mtDNA或Y染色体，这个值往往和谱系的 时间深度 相关。
  • 重要性：树的高度可以用于估计不同族群的遗传分化的时间深度，帮助了解 古老谱系与近代谱系的分化。

4. 分支长度的均值和方差（Mean Branch Length / Var Branch Length）

  • 意义：分支长度代表在树中的遗传差异。平均分支长度反映了不同谱系之间的平均遗传差异，而方差则显示遗传差异的波动性。
  • 重要性：可以用于评估 遗传多样性 和 谱系内的突变速率。特别是在研究mtDNA或Y染色体时，分支长度可以揭示 不同族群的多样性和分化速率。

5. 最近种系距离（MNTD, Mean Nearest Taxon Distance）

  • 意义：每个叶子节点（即样本）的最近邻叶子节点的平均距离。它反映了谱系之间的紧密程度。
  • 重要性：在Y染色体或线粒体DNA的研究中，MNTD可以揭示 不同族群的亲缘关系，帮助识别哪些谱系在遗传上最接近。

6. 系统发育多样性（Phylogenetic Diversity, PD）

  • 意义：基于系统发育树的总分支长度来衡量遗传多样性。较高的PD意味着较高的谱系多样性。
  • 重要性：PD对研究 人口历史事件（如迁徙或隔离） 的影响特别有用，因为它可以反映 群体间的遗传差异 和 多样性丧失。

7. Pigot's Rho

  • 意义：衡量树在不同时间段内的分化速率变化。正值表示分化速率在后期增加，负值表示早期有更高的分化速率。
  • 重要性：适用于研究 谱系扩展模式，例如某些时期的快速人口扩展或迁徙事件。在人类Y染色体或线粒体DNA的研究中，rho值可以帮助我们理解特定人群的 历史人口动态。

8. 四分叉和多分支节点统计量（Pitchforks, Double Cherries, Four-Prong Index）

  • 意义：这些指标衡量系统发育树中有多个叶子节点从同一个祖先节点派生的情况，反映了分支的不均匀性。
  • 重要性：Y染色体和线粒体DNA由于单亲遗传和特定族群的快速扩张，常出现多分支节点。这些指标可以帮助识别 特定时期的大规模扩张事件，如 “Y染色体亚当” 或 线粒体夏娃 相关的族群扩展。

9. Gamma 统计量

  • 意义：衡量内部节点在树中的位置，相对于根的位置。正值表示大多数分支发生在树的晚期，负值表示更多分支发生在树的早期。
  • 重要性：在人类谱系的研究中，Gamma值可以帮助理解 谱系分化模式 和 不同时间点的多样性变化，例如某些族群是否经历了早期或晚期的快速分化。
