准备文件

输入文件可以是 fasta 或者 vcf。

注意

fasta 文件不能够出现任何特殊字符，只能够包含 A,G,C,T。也就是说不能够使用对齐的 fasta 文件。用原始的即可。
VCF 文件分为两种，一种是芯片的，另一种是 WGS 的。如果是芯片的，就要加上一个参数 --chip

代码

VCF 文件芯片

haplogrep-cmd-master/haplogrep classify \
--in input/你的数据.vcf \
--format vcf \
--phylotree 17 \
--extend-report \
--lineage 1 \
--out outcome/1 \
--chip
# --lineage 0代表不生成。1代表生成位点和拓扑结构。2代表生成拓扑结构。选1最好。

VCF 文件全测

haplogrep-cmd-master/haplogrep classify \
--in input/你的数据.vcf \
--format vcf \
--phylotree 17 \
--extend-report \
--lineage 1 \
--out outcome/1 \

fasta 文件

haplogrep-cmd-master/haplogrep classify \
--in input/你的数据.fasta \
--format fasta \
--phylotree 17 \
--extend-report \
--lineage 1 \
--out outcome/1 \

可视化

会生成一个 dot 文件，使用 dot 软件即可。软件下载很简单，linux 系统直接 sudo get 即可。 这个文件被用于可视化：

dot 1.dot \
-Tpdf > 1.pdf

问题

这个软件只能使用 phylotree 17.0 。我不知道怎么使用更新的树。所以我直接把想要的树的内容（从 haplogrep3获得）将原本的文件进行替换。我不知道有没有用。

举例来说：

Haplogrep3/trees/phylotree-rcrs/17.2
Haplogrep3/trees/phylotree-rcrs/17.1
Haplogrep3/trees/phylotree-rcrs/17.0

内部各自有 树.xml 和对应的权重 weghit.txt 文件：

Haplogrep3/trees/phylotree-rcrs/17.2/weights.txt
Haplogrep3/trees/phylotree-rcrs/17.2/tree.xml

分别可以用来替换第二代软件的树文件：

haplogrep-cmd-master/data/phylotree/phylotree17_FU1a.xml
haplogrep-cmd-master/src/main/resources/phylotree17_FU1a.xml
haplogrep-cmd-master/src/main/resources/weights17_FU1a.txt
请一定不要改文件名字，仅替换内容！
