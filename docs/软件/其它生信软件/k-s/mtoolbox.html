1. 安装

使用以下命令将 MToolBox 的路径添加到系统的 PATH 中：

export PATH="/path/to/MToolBox/:$PATH"

在 MToolBox 文件夹中，运行 install.sh 脚本（适用于 Linux 发行版）：

./install.sh

以完成所有 MToolBox 依赖项的完整安装。这可能需要一些时间，请耐心等待，直到整个过程成功完成。我们强烈建议使用安装脚本提供的默认版本：

  • GSNAP 2015-12-31.v7
  • Anaconda distribution 2-2.5.0
  • Zlib 版本 1.2.8
  • MUSCLE 版本 3.8.31
  • samtools 版本 1.3

GSNAP 数据库生成的默认 Kmer 值是 15。不过，如果您想在完整安装过程中更改上述版本或参数，请指定以下选项：

./install.sh -g <gsnap_version> -a <anaconda_version> -z <zlib_version> -m <muscle_file> -s <samtools_version> -k <kmer_to_build_gsnap_db>

要重新安装/更新某个 MToolBox 软件依赖项，请执行：

./install.sh -i <software_name>

其中可能的软件名称包括：

  • gsnap
  • anaconda
  • muscle
  • zlib
  • samtools
  • gsnap_db
./install.sh -i gsnap -g 2016-05-25

2. 系统要求

  • 基于 UNIX 的操作系统
  • MacOS

运行 install.sh 脚本（参见安装部分）时，将默认安装以下所有依赖项：

  • Python 2.7 (www.python.org, 包含在 Anaconda 发行版中, https://www.continuum.io/downloads)
  • GSNAP (http://research-pub.gene.com/gmap/)
  • Samtools (https://sourceforge.net/projects/samtools/files/samtools/)
  • MUSCLE (http://www.drive5.com/muscle/downloads.htm)
  • ZLIB (http://zlib.net/)
  • GATK（可选，仅当用户希望运行 GATK IndelRealigner 时。https://software.broadinstitute.org/gatk/download/）。用户需要将 GATK 包放入 MToolBox/ext_tools 文件夹中： mv GenomeAnalysisTK.jar /path/to/MToolBox/MToolBox/ext_tools/

其他所需文件

默认情况下，MToolBox 采用 RSRS（重建智人参考序列，PMID: 22482806）作为线粒体参考基因组，hg19 作为核参考基因组。或者，用户也可以选择使用 rCRS（修订的剑桥参考序列）。MToolBox 使用的 hg19 核基因组和线粒体 DNA 的 Fasta 文件可在 https://sourceforge.net/projects/mtoolbox/ 上获取，以方便用户使用。然而，在流水线的 v.1.0 版本中，这些文件现在通过 install.sh MToolBox 脚本默认下载和安装，连同所需的 GSNAP 数据库。

MToolBox 文件夹包含 MITOMAP_HMTDB_known_indels.vcf 文件，其中包含 127 个在 MITOMAP 和 HmtDB 中注释的已知插入缺失变异，以及 GATK 的 GenomeAnalysisTK.jar 模块使用的相关 intervals_file.list。MToolBox 文件夹还包括两个以制表符分隔的文件，patho_table.txt 和 sitevar_modified.txt，分别包含在注释步骤中使用的变异特异性和位点特异性信息。

文件命名注意事项

对于每个样本，输出文件夹和文件的基本名称将从输入文件名中解析。

  • BAM/SAM 文件：BAM 或 SAM 文件必须重命名为 <sample_name>.ext，例如：
mv HG00096.chrom20.ILLUMINA.bwa.GBR.low_coverage.20101123.bam HG00096.bam

这样“HG00096”将成为输出的基本名称。

  • FASTQ 文件：对于配对末端数据，FASTQ 文件必须重命名为 <sample_name>.R1.fastq 和 <sample_name>.R2.fastq；对于单端数据，必须重命名为 <sample_name>.fastq。FASTQ 压缩输入文件可以接受 *.fastq.gz 扩展名。

重要：请注意，MToolBox 每个样本只能识别一对配对末端的 fastq 文件（R1+R2）和一个单端 fastq 文件。

3. 使用

MToolBox.sh 选项：

  • -i 配置文件 [强制]
  • -m mapExome 脚本的选项 [参见 mapExome.py -h 了解详情]
  • -a assembleMTgenome 脚本的选项 [参见 assembleMTgenome.py -h 了解详情]
  • -c mt-classifier 脚本的选项 [参见 mt-classifier.py -h 了解详情]

帮助选项：

  • -h 显示此帮助信息
  • -v 显示版本信息

用法：

#此命令将在 test_config.sh 文件中指定的样本集上运行 MToolBox，将 GSNAP 线程数设置为 20 (-m "-t 20..")，即从读取端允许保留 indel 的最小核苷酸距离为 10 （-a“-t 10”）和在 FASTA 共有序列中保留核苷酸变异的异质性阈值 0.6（-a“-z 0.6”）。
MToolBox.sh -i test_config.sh -m "-t 20" -a "-t 10 -z 0.6"

4. 输出文件

MToolBox 的默认输出包括：

  • VCF_file.vcf：包含所有与 RSRS/rCRS 比较的线粒体变异位点和其他元信息。
  • mt_classification_best_results.csv：报告每个序列的最佳单倍群预测结果。如果排序结果中有多个具有相同概率的最佳单倍群预测，输出将包含所有这些预测。
  • prioritized_variants.txt：仅包含每个样本分析的优先变异的注释，这些变异是由三个参考序列（rCRS、RSRS 和 MHCS）识别的，并按核苷酸变异性递增排序。
  • summary_<date_time>.txt：报告所选选项的简要摘要、预测的单倍群、每个样本的总变异和优先变异数量，以及仅针对 NGS 数据的重建基因组覆盖率、同质性和异质性变异数量。
  • OUT_<sample_name> 文件夹包含：
    • outmt.sam：映射到人类线粒体 DNA 的读取；
    • logmt.txt：线粒体 DNA 映射的 GSNAP 日志文件；
    • outmt.fastq：从 outmt.sam 文件中提取的单端读取的 fastq 文件；
    • outmt1.fastq：从 outmt.sam 文件中提取的配对读取的 fastq 文件；
    • outmt2.fastq：从 outmt.sam 文件中提取的配对读取的 fastq 文件；
    • outhumanS.sam：映射到整个基因组的人类基因组的单端读取；
    • loghumanS.txt：人类基因组映射（单端读取）的 GSNAP 日志文件；
    • outhumanP.sam：映射到整个基因组的人类基因组的配对读取；
    • loghumanP.txt：人类基因组映射（配对读取）的 GSNAP 日志文件；
    • OUT.sam：唯一映射到线粒体基因组的读取对齐；
    • OUT2.sam：在经过 IndelRealigner 和/或 MarkDuplicates 处理后唯一映射到线粒体基因组的读取对齐。即使禁用这两个过程，也会生成此文件；
    • mtDNAassembly-table.txt：描述组装位置的主要表格；
    • mtDNAassembly-Contigs.fasta：包含所有重建 contig 的 fasta 文件；
    • mtDNAassembly-coverage.txt：包含每个 contig 和已知线粒体注释的覆盖率的文本文件；
    • logassemble.txt：assembleMTgenome.py 脚本的日志文件；
    • sorted.csv：包含预测概率超过 90% 的每个单倍群的表格。包括以下字段：
      1. N = 片段序列相对于 RSRS 的 SNP 数量；
      2. Nph = 在 Phylotree 中映射的 SNP 数量（在 N 中）；
      3. Nph_tot = 整个基因组中定义单倍群的 SNP 数量；
      4. Nph_exp = 片段区域中定义单倍群的 SNP 数量；
      5. P_Hg = 单倍群的预测百分比值（Nph/Nph_exp*100）；
      6. Missing sites = 在查询基因组中不存在但预期在其相应路径上的变异。这些变异也可能指向测序错误；
    • merged_diff.csv：报告查询基因组与三个序列（RSRS、rCRS 和 hg_MHCS（Macro-Haplogroup Consensus Sequence））之间的 SNP；
    • <sample_name>.csv：包含表格，其中报告了 Phylotree Build 15 中所有单倍群的相同数据，如 <sequence_name>.sorted.csv 文件，但不包括 Missing Sites 字段；
    • annotation.csv：对 merged_diff.csv 文件的进一步解释，为每个 mt 变异等位基因提供多个注释，查询基因组和三个序列（RSRS、rCRS 和 hg_MHCS）之间：
      • Sample = 样本名称；
      • Variant Allele = 线粒体基因组中的核苷酸位置及变异等位基因；
      • HF = VCF 文件中报告的异质性比例；
      • CI_lower;CI_upper = 异质性比例置信区间的上下限；
      • RSRS = 如果为“是”，则变异被 RSRS 识别；
      • MHCS = 如果为“是”，则变异被 Macro-Haplogroup Consensus Sequence 识别；
      • rCRS = 如果为“是”，则变异被 rCRS 识别；
      • Haplogroup = 最佳预测单倍群；
      • Other Haplogroups = 如果为“+”，则该变异定义了其他单倍群；
      • Locus = 线粒体基因座；
      • Nt Variability = SiteVar 变异性值；
      • Codon Position = 密码子内的核苷酸位置；
      • Aa Change = 氨基酸变化；
      • Aa variability = MitVarProt 氨基酸变异性值；
      • tRNA annotation = 线粒体 tRNA 基因的特定信息（在 tRNA 中的位置；tRNA 类型；三叶草二级结构区域；成熟核苷酸；特定位置在 tRNA 折叠中的作用）；
      • Disease score = 综合疾病评分，作为非同义变异致病性预测评分的加权平均。详情参见相关出版物（PMID: 26621530）；
      • RNA predictions = rRNA 基因中的 49 个变异和 tRNA 基因中的 207 个变异的评分（PMID: 24092330；PMID: 21882289；PMID: 23696415）。评分在 0 到 1 的范围内。rRNA 的阈值为 0.51，tRNA 的阈值为 0.31。低致病性低于固定阈值；
      • MutPred pred = MutPred 预测（高致病性，低致病性）；
      • MutPred Score = MutPred 致病性评分（0.000-1.000）；
      • PolyPhen-2 HumDiv Pred = Polyphen-2 HumDiv 预测（良性，可能有害，可能有害，未知）；
      • PolyPhen-2 HumDiv Prob = Polyphen-2 HumDiv 概率（0.000-1.000）；
      • PolyPhen-2 HumVar Pred = Polyphen-2 HumVar 预测（良性，可能有害，可能有害，未知）；
      • PolyPhen-2 HumVar Prob = Polyphen-2 HumVar 概率（0.000-1.000）；
      • PANTHER Pred = 通过 SNPs&GO 软件进行的 PANTHER 预测（中性，疾病，未分类）；
      • PANTHER Prob = 通过 SNPs&GO 软件进行的 PANTHER 概率（0.000-1.000）；
      • PhD-SNP Pred = 通过 SNPs&GO 软件进行的 PhD-SNP 预测（中性，疾病，未分类）；
      • PhD-SNP Prob = 通过 SNPs&GO 软件进行的 PhD-SNP 概率（0.000-1.000）；
      • SNPs&GO Pred = 通过 SNPs&GO 软件进行的 SNPs&GO 预测（中性，疾病，未分类）；
      • SNPs&GO Prob = 通过 SNPs&GO 软件进行的 SNPs&GO 概率（0.000-1.000）；
      • Mitomap Associated Disease(s) = MITOMAP 疾病相关突变注释；
      • Mitomap Homoplasmy = MITOMAP 同质性状态注释；
      • Mitomap Heteroplasmy = MITOMAP 异质性状态注释；
      • Somatic Mutations = MITOMAP 体细胞突变的细胞或组织类型注释；
      • SM Homoplasmy = 体细胞突变的同质性状态注释；
      • SM Heteroplasmy = 体细胞突变的异质性状态注释；
      • ClinVar = ClinVar 疾病相关突变注释；
      • OMIM Link = OMIM 条目链接；
      • dbSNP ID = dbSNP 数据库中的 rs ID；
      • Mamit-tRNA link = Mamit-tRNA 站点注释链接；
      • PhastCons20Way = 使用 hg38+rCRS 作为参考序列计算的 20 个脊椎动物的 PhastCons 保守性评分；
      • PhyloP20Way = 使用 hg38+rCRS 作为参考序列计算的 20 个脊椎动物的 PhyloP 保守性评分；
      • AC/AN 1000 Genomes = 在 1000 Genomes 中找到的可能致病变异的等位基因计数与等位基因数量的比率；
      • 1000 Genomes Homoplasmy = 1000 Genomes 变异中的同质性状态注释；
      • 1000 Genomes Heteroplasmy = 1000 Genomes 变异中的异质性状态注释。

警告：请注意，异质性比例和相关的置信区间仅会报告针对选择的参考序列进行读取映射的变异。

5. 处理 MToolBox 数据的辅助脚本

这些脚本位于 MToolBox 的 aux 文件夹中。

  • get_prioritized_var_and_summary.sh 是一个辅助脚本，用于从 MToolBox 的 annotation.csv 文件中生成 summary.txt 和 prioritized_variant.txt 文件。
    运行以下命令以获取脚本使用帮助：
    ./get_prioritized_var_and_summary.sh
  • filter_HF.py 是一个辅助脚本，用于过滤 MToolBox VCF 文件（版本 < 1.2 生成的文件）的 HF 和 DP，并将多等位基因 SNP 分割为每行一个。
    python filter_HF.py
  • filter_HF_v2.py 是一个辅助脚本，用于过滤 MToolBox VCF 文件（版本 >= 1.2 生成的文件）的 HF 和 DP，并将多等位基因 SNP 分割为每行一个。
    python filter_HF_v2.py

使用示例

python filter_HF_v2.py HG00119 HG00119.vcf 0.8 100 txt HG00119.txt No
