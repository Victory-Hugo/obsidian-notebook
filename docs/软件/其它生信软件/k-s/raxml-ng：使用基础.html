这是一个预编译的版本，Linux 应该直接可以使用：raxml-ng

基本用法：

命令行输入 raxml-ng [选项] 来运行程序。

主要命令（互斥）：

  • --help：显示帮助信息。
  • --version：显示版本信息。
  • --evaluate：评估一棵树的似然值（含模型和枝长优化）。
  • --search：最大似然树搜索（默认：10棵拟合树和10棵随机起始树）。
  • --bootstrap：自动检测重复次数的Bootstrap分析。
  • --all：一体化分析（ML树搜索 + Bootstrap）。
  • --support：对一棵参考树（如最佳ML树）和一组重复树（如Bootstrap分析所得）计算双分支支持。
  • 更多命令及详细参数见使用说明。

输入输出选项：

  • --tree：指定起始树，可以是随机、拟合或指定的新疆文件。
  • --msa：指定比对文件。
  • --msa-format：指定比对文件格式，如FASTA、PHYLIP等。
  • --data-type：指定数据类型，如DNA、氨基酸等。
  • 更多选项如输出文件前缀、日志详细程度、精度等也可设置。

模型选项：

  • --model：模型设置或指定分区文件。
  • --brlen：分区间枝长的关联性。
  • --opt-model：是否优化所有模型参数。
  • --opt-branches：是否优化所有枝长。
  • 更多模型相关的设置包括优化方法、似然值精度等。

举例

使用 GTR+G 模型建树

./raxml-ng --msa testDNA.fa --model GTR+G

一步到位

./raxml-ng --all --msa testAA.fa --model LG+G8+F --tree pars{10} --bs-trees 200

# 进入工作目录
cd Raxml-ng/

# 运行 RAxML-NG 构建最大似然树
raxml-ng --all \
         --msa trim_out.fasta \
         --model GTR+G+I \
         --threads 32 \
         --bs-trees 20 \ # 生成 20 个 Bootstrap 树以评估树的稳健性。
         --prefix ml_tree
