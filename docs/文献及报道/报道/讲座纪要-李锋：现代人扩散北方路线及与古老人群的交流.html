“现代人”的概念

讲座纪要 | 李锋：现代人扩散北方路线及与古老人群的交流 (qq.com)
2024年5月10日，北京大学考古文博学院长聘副教授李锋老师在立德楼305为人大考古师生带来《现代人扩散北方路线及与古老人群的交流》的精彩讲座。

“现代人”（anatomically modern humans）即学术上的智人种（Homo sapiens）。

  1. 从解剖学意义上而言，现代人的体质特征与当今的现生现代人体格更加接近。不过需要注意的是，化石研究和生物学上关于人类“种”的概念有一定的差别，且应当理解人类不同“种”之间还存在着基因交流。
  2. 人类除在体质特征之外，更重要的是文化特征/技术特征，这种文化意义上的现代人通过石叶技术、石器标准化和多样化、复杂用火、交换、大型动物利用、季节性移动、极端环境的开发等“现代人行为”所定义，这些“行为清单”随着研究的深入不断更新，“清单”中人类的行为可能会发生渐变或跃变。
  3. 随着近年古基因研究的深入，“现代人”的另外一层意义是遗传学意义的现代人，以付巧妹老师团队研究的田园洞人为例，虽然其具有东亚人群的特征，但并非是我们最直接的祖先，这表明并非每一例早期现代人个体都对当今人类有直接的基因贡献。

关于现代人起源的假说，大致可分为三种，即

  1. Wolpoff教授、Thorne教授和吴新智院士等学者提出的“多地区进化”假说，强调不同地区人类连续进化和区域之间的交流；
  2. Cann等以及Stringer等学者的“出自非洲”假说,即完全替代说、“夏娃假说”等，认为距今20万年左右非洲出现的现代人将其它区域的古人类完全替代；
  3. 以Smith F等学者为主提出的“同化吸收”假说，认为非洲走出的现代人为主，在西亚、欧洲等地与当地的尼安德特人发生基因交流。随着新近古DNA研究，不同人群（尼安德特人、丹尼索瓦人，早期现代人等）发生基因交流，出自非洲的“完全替代”（total replacement）假说基本被摒弃。
  4. 吴新智院士在多地区进化假说的基础上，基于中国发现的古人类化石特征，指出中国地区的早期现代人应当呈现的是一种区别于旧大陆西侧的、带有连续性的模式，提出“连续演化，附带杂交”的表述；
  5. 高星等学者对现代人起源假说进行总结，提出现代人演化可能具有“区域多样性模式”。

有关现代人起源，哪种假说更加合理还没有明确的答案，直立人与现代人之间是否存在年代断层或是否存在遗传联系也都存在争议，这些问题还需要未来数据的支撑。不过学术界已达成的共识是最早人类起源于非洲；尼安德特人等古老型人群和现代人之间有基因交流，如2010年对尼安德特人和丹尼索瓦人两种古老型人类的全基因组测序，从分子生物学角度揭示了这两个现代人“近亲”的遗传信息。总之，虽然有关问题还需要持续推进，但是可以明确的是，现代人的形成过程中是存在广泛的交流与扩散的。

现代人向东亚的扩散

南线

关于这一问题的研究以往国际上多关注南方路线。学术界认为，早期现代人从非洲向东亚进发，走了一条“南方路线”，这条路线途径阿拉伯半岛、印度、东南亚等地。在南方路线的讨论中，缺乏在非洲之外足够的人类化石证据，仅有阿拉伯半岛发现可能为现代人特征的化石，另外为此路线提供支撑的还有该半岛上发现的距今约12万年的人类足迹化石，根据足迹化石推断这是属于体质特征为早期现代人的遗存。由于人类化石材料的不足，南方路线的支撑主要集中在石制品遗存上，包括两组主要的证据，即石器时代中期（Middle Stone Age）和几何形细石器遗存，不过目前中国还未发现中期（MSA/MP）技术特征的明确石制品组合，仅有少量线索。相比之下，北方路线却有明确技术扩散的考古证据。

北线

现代人向东亚扩散的北方路线证据主要体现在石器技术、人类化石及古DNA上。石器技术是论证早期现代人扩散路线的重要证据，旧石器时代晚期初段（Initial Upper Paleolithic）石器技术的扩散是早期现代人北方扩散的结果。保加利亚Bacho Kiro Cave发现年代为约4.5万年的现代人化石和IUP石制品组合，基因研究发现此遗址的现代人与尼安德特人有较为频繁的基因交流。距今4.1-3.4万年的水洞沟第1地点的石制品具有欧亚大陆西方旧石器时代中晚期过渡的特点，对该地点的石制品遗存分析，并与西伯利亚阿尔泰地区、贝加尔地区相关IUP遗址石制品技术进行对比显示，水洞沟第1地点的石器技术与西伯利亚阿尔泰地区更为接近，其与年代更早的西伯利亚地区的同类发现存在着技术联系。不过，通过IUP石叶技术构建的“北方路线”依然存在着一些考古材料上的空白：如中亚地区由于考古工作强度尚且不足，还缺少足够的证据等。

沿北方路线扩散的现代人与古老类型人类存在着复杂的人群交流，例如尼安德特人、丹尼索瓦人和现代人三者相互之间都有着频繁的基因交流。甘肃夏河遗址有采集到丹尼索瓦人的基因，但该遗址的石器技术发布的材料还不多，暂时还无法与丹尼索瓦洞的石器技术相对应。学术界认为尼安德特人通常与莫斯特技术共生，随着近年来对金斯太遗址的发掘和研究，在距今约4.7-3.7万年的层位发现了与莫斯特技术体系相近的石器组合。此外，通天洞遗址距今4.6-4.4万年的层位以盘状石核为主的遗存同样有莫斯特石器组合。从技术意义上而言，这些材料为讨论尼安德特人的扩散及与早期现代人的交流提供了非常重要的线索。

在讲座的最后，李锋老师对讲座内容进行小结：现代人向欧亚大陆东部扩散的南方路线和北方路线共存。沿中亚、西伯利亚、蒙古、中国西北地区等向东扩散的北方路线（MIS 3）要晚于南方路线（MIS 5）；现代人的扩散是一个复杂过程，其中既有“成功”的扩散也有“失败”的扩散；现代人扩散的北方路线更为复杂、进行了多次尝试，过程中与尼安德特人、丹尼索瓦人等古老型人类有过基因交流。“北方扩散路线”表明现代人的扩散是一个多时段、多路线、复杂的动态过程。

讲座结束后，李老师和在场的师生们对西藏尼阿底、河北泥河湾新庙庄遗址、西伯利亚阿尔泰地区考古遗址的石器技术特征及现代人起源的假说模型进行了交流和讨论。讲座在掌声中落下帷幕。
