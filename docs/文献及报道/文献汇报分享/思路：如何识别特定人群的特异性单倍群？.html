前言

这个问题说简单也简单，说复杂也复杂。简单之处在于，从遗传学的角度来看，单倍群的分化历史是固定的，尽管各种估算时间的方法都存在一定的局限性（ρ（rho）方法、贝叶斯进化树分析（Bayesian Evolutionary Analysis by Sampling Trees，BEAST）），但是整体来说是基本稳定的。复杂之处在于，人群的迁徙与混合（民族、文化属性）并不一定与单倍群的迁徙与混合（遗传属性）匹配。因此，这衍生出来群体遗传学（或者叫做分子人类学）的极其重要的研究议题：文化扩散(cultural diffusion) 和人口扩散(demic diffusion)。

找到某个特定人群的单倍群目前来看是解决上述问题的关键。尽管没有一个全体认可的、统一的单倍群定义标准，多个研究已经从不同角度探讨了如何定义特定区域的人群单倍群。

例如，在 2017 MBE 撒丁岛人的线粒体基因组多样性研究中，研究者是这样定义的：

  • 利用如下条件判断撒丁岛特有的单倍群（区域特异性的单倍群）：
    • 只包含撒丁岛起源的线粒体 DNA：该单倍群仅由撒丁岛起源的 mtDNA 组成。
    • 至少由三个线粒体基因组定义，并包含至少两个单倍型：这个单倍群必须包括至少三个线粒体基因组，并且这些基因组至少存在两个单倍型（即至少有一个同质性突变的差异）。
    • 在其根部包含至少一个稳定突变：该单倍群的根部必须有至少一个稳定突变（即在系统发育树中不重复出现的突变）。这意味着该突变在进化过程中保持不变，表明该单倍群在撒丁岛独立分化。

另外的一篇研究 2019 NSR 新石器时代的粟（作）农（民）通过采用大麦农业为青藏高原的永久定居做出了贡献做出了类似的定义：

识别藏族中独立分化的单倍群 ：

  1. 分支主要由藏族样本占据：这表明这些线系或分支在藏族中有明显的存在，而且在遗传上可能表现出与其他人群的显著不同。
  2. 其他地区的样本在这些线系/分支中分布零星：这种分布模式通常反映了从青藏高原向这些区域的近期基因流动。
    具体来说，我们这样理解：
  • 核心问题：研究人员想要探讨青藏高原的西藏人群是否可以识别出独立分化的单倍群。这背后的假设是，大约在3.6千年前，种植粟类的农民迁移到了青藏高原，他们的遗传特征应当在今天的西藏人中保留并进一步独立分化。
  • 假设的逻辑：
    1. 如果这些农民真的迁移过来，他们的后代应该在当代西藏人中留下遗传印记。
    2. 由于时间的推移，这些遗传印记（单倍群）可能会与其他群体发生新的遗传分化，形成新的特异性单倍群。
      例子：可以类比为，假如一群人搬到一个岛上，过了几千年，岛上的人不仅保留了部分最初迁移者的特征，还由于环境和时间的推移，发展出自己独特的新特征（分化）。
    • 挑战：要识别哪些单倍群是新分化出来的，以及它们是否与3.6千年前的迁徙有关，研究者需要通过年龄估算来确定这些单倍群的起源。
    • 年龄估算的困难：如果只看西藏个体的某个单倍群，会引入偏差。原因是即使在同一个单倍群中，不同的西藏个体可能位于不同的分支上。这意味着这些个体的基因谱系虽然有共同的祖先，但在时间上分化较早或较晚，因此不能简单地通过西藏人群中的分支年龄来确定单倍群的整体年龄。
  • 具体分析步骤：
    1. 第一标准：找到那些主要由西藏样本占据的单倍群分支。也就是说，这些单倍群的分支在当代西藏人中是主要存在的，说明它们在西藏的遗传背景中扮演了重要角色。
    2. 第二标准：其他地区的样本在这些单倍群中的分布应该是零星的。这意味着，这些单倍群分支主要是在青藏高原上形成的，其他地区的样本可能是由于最近的基因流动带来的。

结合这些观点，我总结为如下几个要点。

识别标准

以识别黄河流域的粟（millet）作农业人群的相关单倍群为例：

1. 时间一致性标准

标准内容：

单倍群的分化时间应与黄河流域粟作农业的出现和扩展时间相一致，具体为距今7,000-4,000年之间。粟作农业最早出现在新石器时代早期的黄河流域地区，如裴李岗文化、仰韶文化和龙山文化，因此相关单倍群的共祖时间应与这一时期重合。

具体操作：

  • 使用古代DNA样本和现代样本，通过共祖时间估算（如使用BEAST或BATWING）推测单倍群的分化年龄。
  • 确保所识别的单倍群或其特定分支的共祖时间（tMRCA）在7,000-4,000年的时间窗口内。

举例：

  • 如果某个单倍群的共祖时间约为5,000年前，并且样本集中在黄河流域，这个单倍群可能与粟作农业扩展相关。

2. 地理分布集中性标准

标准内容：

单倍群的地理分布应在黄河流域及其周边地区有较高的集中度，尤其是在与粟作农业相关的新石器时代文化遗址周围。该单倍群在现代北方中国居民中的分布应具有明显的黄河流域聚集性，并且在其他区域的分布较为零星。

具体操作：

  • 结合系统地理分析，分析该单倍群或其分支在黄河流域的分布。
  • 比较现代北方人群与周边地区（如长江流域、内蒙古等）的样本分布情况，确定黄河流域的地理集中性。
  • 搜索古代粟作农业遗址（如裴李岗、仰韶、龙山文化遗址）中出现的相关单倍群。

举例：

  • 假设某单倍群主要分布在河南、山西和陕西等地区，而在南方地区只存在少量零星个体，这可以表明该单倍群可能与黄河流域的早期农业扩展有关。

3. 谱系独立分化标准

标准内容：

单倍群中的特定分支应在黄河流域地区表现出独立的de novo分化特征，即该分支主要由黄河流域的个体占据，而在其他地区（如东亚其他地区）只出现零星的样本。这意味着这些分支在黄河流域经历了独立的遗传分化，而非从其他地区引入。

具体操作：

  • 构建该单倍群的系统发育树（如使用MEGA、BEAST或MrBayes），观察不同分支的地理分布。
  • 确保黄河流域的样本在该单倍群的分支上具有较高的占比，且与其他地区样本的谱系分布存在显著差异。
  • 通过网络分析（如Network软件）进一步细分该单倍群，识别在黄河流域内的特定分化分支。

举例：

  • 假设单倍群O中的某个分支“O2b”主要由黄河流域的样本占据，并且在其他地区仅有零星的分布，可以推断该分支可能在黄河流域独立分化。

4. 文化与遗传相关性标准

标准内容：

该单倍群应与已知的粟作农业文化遗址和考古证据相匹配。具体而言，应在裴李岗、仰韶、龙山等粟作农业遗址中发现与该单倍群相关的古代DNA样本，并且这些样本的遗传特征应与现代黄河流域人群中的相应单倍群保持一致。

具体操作：

  • 搜集古代DNA样本，尤其是来自粟作农业相关考古遗址的遗传数据，并与现代样本进行比对，识别出共同的单倍群。
  • 将这些考古遗址中的遗传数据与现代黄河流域的遗传数据结合，使用PCA或ADMIXTURE等聚类分析工具，检测古代样本和现代黄河流域人群之间的遗传连续性。

举例：

  • 如果在仰韶文化遗址中发现了属于单倍群O的古代样本，并且现代河南和陕西居民中也发现了同样的单倍群O，这可能证明该单倍群与粟作农业扩展有密切关系。

5. 基因流动限制性标准

标准内容：

该单倍群或其分支应显示出有限的跨区域基因流动。也就是说，黄河流域的粟作农业人群应在该单倍群的遗传谱系中表现出显著的遗传隔离，表明该支系是本地农业扩展的结果，而不是通过大规模迁徙或基因流动引入的。

具体操作：

  • 使用迁移模型分析工具（如MIGRATE-N或IMa2）检测黄河流域与其他地区（如长江流域、内蒙古或东亚其他地区）之间的基因流动情况。
  • 确保该单倍群或其分支在黄河流域表现出较少的外来基因流动，表明其在黄河流域的本地起源。
  • 通过FST分析测量不同地区之间的遗传分化，确保该单倍群在黄河流域具有相对独立的遗传特征。

举例：

  • 假设通过基因流动分析发现该单倍群在黄河流域的流动性有限，且与南方地区或其他东亚地区的基因交流较少，那么可以推断该单倍群可能与本地的粟作农业扩展密切相关。

总结 ：明确的标准体系

为了寻找与黄河流域粟作农业相关的单倍群，可以制定以下五个标准：
时间、空间、文化、独立、

  1. 时间一致性标准：单倍群的分化时间应与黄河流域粟作农业出现的时间（距今7,000-4,000年）一致。
  2. 地理分布集中性标准：该单倍群应在黄河流域及其周边地区有较高的集中度，其他地区零星分布。
  3. 谱系独立分化标准：该单倍群或其特定分支应在黄河流域表现出独立分化的特征，主要由黄河流域样本占据。
  4. 文化与遗传相关性标准：该单倍群应与粟作农业相关的考古遗址和文化遗址相匹配，具有遗传和文化上的联系，最直观的是古代 DNA 证据的支持。
  5. 基因流动限制性标准：该单倍群应显示出有限的跨区域基因流动，表明其遗传特征在黄河流域是本地的而非通过外来基因流动引入的[1]。

尾注

  1. “基因流动限制性标准”主要用于确认某个单倍群是否在特定的时间和区域内，尤其是研究的早期阶段，与某一特定文化（如黄河流域的粟作农业）有强关联。这个标准帮助我们排除那些通过外来基因流动引入的单倍群，因为我们想证明某个单倍群在黄河流域的独立发展，而不是从其他地区迁入。但是，这并不意味着这个单倍群在之后的历史中不能扩散到其他区域。实际上，很多单倍群最初在某个地方形成，随后伴随人群迁徙传播到其他地区。这种扩散并不削弱它最初与黄河流域粟作农业的关联。︎
