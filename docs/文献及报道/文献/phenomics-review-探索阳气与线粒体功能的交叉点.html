摘要

中医的阳气概念与现代医学的线粒体有很多相似之处。两者对于人类来说都是不可或缺的，与生死密切相关。本文讨论了线粒体与阳气在体温、衰老、新生儿、昼夜节律、免疫、经络等各个方面的相似之处。众所周知，阳气对于人体健康至关重要。

有趣的是，线粒体功能下降被认为是各种疾病发生的关键。研究人员从线粒体功能的角度进一步阐述阳气不足引发的疾病，如癌症、慢性疲劳综合症、睡眠障碍、老年痴呆、代谢性疾病等。目标是建立两个重要概念之间的相似性和联系，并希望我们的文章能够激发对统一西医和替代医学，特别是中医的重要概念的进一步讨论和研究，并为理解人类健康提供独特的整体见解。

介绍

线粒体的生理

  1. 线粒体DNA（mtDNA）：

    • 线粒体含有自己的基因组，称为线粒体DNA（mtDNA），这是一种环状的双链分子。
    • 人类mtDNA由16,569个碱基对组成，编码37个基因：13个蛋白质，22个转运RNA和2个核糖体RNA。
    • 这13个蛋白质是氧化磷酸化（OXPHOS）酶复合体I、III、IV和V的关键亚单位。

  2. 遗传特性：

    • 人类mtDNA是母系遗传的，因为父系mtDNA在受精后立即被主动破坏。
    • 与核DNA（nDNA）不同，mtDNA没有被组蛋白保护，也不在减数分裂期间进行重组。
    • mtDNA的突变率比nDNA高大约10至100倍。

  3. 线粒体的主要功能：

    • 线粒体的主要功能是生成大量能量。
    • 作为代谢信号中心，线粒体可以利用三羧酸循环（TCA循环）和电子传递链（ETC）将碳水化合物、蛋白质和脂肪分子转化为CO2和水，同时产生三磷酸腺苷（ATP）。
    • 在线粒体中，底物氧化与ATP形成的耦合被称为OXPHOS，这对各种组织和器官至关重要。

  4. 其他生理功能：

    • 除了能量生产，线粒体在多个细胞过程中也至关重要，例如凋亡、脂肪酸的β-氧化、细胞代谢产物生成、活性氧（ROS）信号传递、类固醇合成和钙离子稳态。
    • 近期研究显示，线粒体还是表观遗传学和免疫反应的中心调节器。

线粒体与疾病

  1. 线粒体功能失调的特征：

    • 线粒体功能失调表现为电子传递链（ETC）效率降低和ATP合成减少。
    • 这种功能失调与许多人类疾病有关，包括神经退行性疾病（如阿尔茨海默病和帕金森病）、心血管疾病、代谢综合征、癌症、肥胖和过早衰老。

  2. 线粒体相关疾病的病因：

    • 这些疾病与多种线粒体相关因素有关，包括mtDNA的删除/突变、ATP缺乏、活性氧（ROS）的增加、线粒体通透性转换孔开放、凋亡、Ca2+调节失常、炎症和融合/分裂动力学的改变。
    • mtDNA突变可导致细胞能量转换严重损害和组织功能失调，由mtDNA突变引起的线粒体疾病包括Leigh综合症、Leber遗传性视神经病变、MELAS（线粒体脑病、乳酸血症和类中风发作）、MERRF（肌阵挛性癫痫伴红色纤维斑）、糖尿病和耳聋。

  3. 传统中医（TCM）的视角：

    • 传统中医的研究大多记录在古代汉语文献中，其方法和观点与西方医学截然不同，难以理解。
    • 传统中医治疗疾病注重整体方法，这被认为是传统中医的典型特征和优势。
    • 例如，传统中医在治疗慢性疲劳综合症、代谢综合征、贫血和哮喘等复杂疾病时，充分利用其整体调节身体的优势。
    • 在古代中国，人们相信“气”是所有生物最基本的动力，维持其功能活动。气与生命一同产生，从幼年到老年，然后从强到弱，最终走向死亡。

中医的“阴阳”

  1. 阴阳理论：

    • 传统中医认为各种疾病是由阴阳内部失衡引起的。通过纠正阴阳失衡，人体可以恢复健康状态。

  2. 《黄帝内经》的重要性：

    • 《黄帝内经》，包括《素问》和《灵枢》，是中国最古老的医学教科书之一。
    • 在《黄帝内经》中，阳被比作阳光，阴被比作水。没有阳光，地球上就没有生命；同理，没有阳，人类无法生存。

  3. 阳气的功能和重要性：

    • 阳气在性质上类似于太阳。如果阳气运行异常，寿命将会缩短。
    • 阳气是人体的关键基本物质，其基本功能包括生产和运输气血和其他体液、维持人体温度、生成和增加能量、加强免疫系统、预防各种疾病、增强新陈代谢和促进循环。

  4. 阳气不足引起的疾病：

    • 阳气不足可能导致多种疾病，如疲劳、失眠、月经痛、头痛、减肥困难，甚至癌症。
    • 导致阳气不足的主要原因包括社会和环境影响、不良生活习惯、疾病（如哮喘）和长期使用抗生素。

  5. 进一步研究的需求：

    • 关于阳气的本质和特性仍需进一步研究。

线粒体与阳气的相似之处

  1. 体温调节：

    • 阳气在传统中医中负责维持体温，而线粒体通过其产生的能量帮助调节体温，表现为热量的产生和分配。

  2. 衰老过程：

    • 在传统中医中，阳气的减少与衰老相关联，被视为生命力的减弱。线粒体的功能衰退也与细胞衰老和相关疾病的发展紧密相关。

  3. 新生儿：

    • 阳气被认为是从出生开始就存在的生命力。在现代医学中，线粒体为新生儿提供必要的能量，支持其快速的生长和发展。

  4. 昼夜节律：

    • 阳气的流动被认为与日夜更替相符，影响身体的节律活动。线粒体活动也显示出昼夜节律性变化，影响能量的产生和消耗。

  5. 免疫系统：

    • 在传统中医中，阳气强健被认为能够抵御外来的病原体。线粒体在调节免疫反应中发挥重要作用，包括能量供应和调控细胞死亡路径。

  6. 经络系统：

    • 经络系统在中医中用于描述体内能量（气）的流动路径，阳气在这些路径中流动以维持身体健康。虽然现代医学中没有直接对应的结构，但线粒体在细胞间的能量转移和信号传递中发挥类似的基础功能。

体温

  1. 阳气不足的表现：

    • 《素问》记载，当人体阳气充足时体温会升高，阳气不足时体温会降低。
    • 阳气不足常表现为体温下降，称为“虚寒”症状，如感觉寒冷、疲劳等，这与阳虚综合症相似，治疗方法是补充阳气。

  2. 动物模型中的观察：

    • 在实验动物（如大鼠）模型中，阳气不足与严重的葡萄糖和脂质代谢障碍相关，显示出能量供应不足的迹象。
    • 其他动物模型研究发现，代谢能量分子（如乳酸和乙酰肉碱）的代谢在阳气不足时也有不同程度的损害或异常。

  3. 线粒体功能与体温：

    • 线粒体的基本功能之一是产生热量并维持体温，体温的来源是线粒体的解偶联，由解偶联蛋白（UCPs）介导。
    • 解偶联蛋白1（UCP1）参与了人类祖先适应寒冷环境的进化过程，寒冷环境诱导UCP1表达，从而在线粒体中产生更多热量。

  4. 线粒体功能失调与体温调节：

    • 如果线粒体的代谢功能受损（例如，线粒体不能高效地将糖和氧转化为能量），可能会导致产热不足，类似于阳气不足时的低体温症状。

衰老

传统中医中《素问》记载，阳气与生命的各个阶段——出生、成长、成熟、衰老和死亡——密切相关，对人类生命周期具有重大意义。阳气的耗尽是衰老的主要原因。《灵枢》指出，人到四十岁时，阳气不足，导致生命力和身体机能的显著下降。这意味着人的一生是一个阳气衰减的过程。中医理论认为，阳气不足是衰老和老年病的主要原因。因此，通常使用一些益气的中草药来缓解与年龄相关的疾病。

类似于阳气，线粒体功能随着年龄增长而衰退，导致氧化磷酸化（OXPHOS）效率下降。线粒体相关的功能障碍（例如与年龄相关的mtDNA突变积累和mtDNA拷贝数下降）被认为是与年龄相关疾病的重要因素。长期使用五味子素B可以减轻与年龄相关的线粒体功能障碍。随着年龄的增长，线粒体ATP产生率（MAPR）每十年约下降8%，线粒体复合体I和IV的活性在灵长类动物大脑皮层中逐渐减弱，导致细胞衰老，特别是在心肌中。mtDNA的突变随年龄在多种组织中累积，导致呼吸链功能急剧下降，随着突变mtDNA的增多，细胞能量降低至低于正常细胞和组织功能所需的最小输出，导致器官功能下降和衰老表型。

新生儿

根据中医理论，每代新生儿出生时阳气会复苏，而新生儿的阳气不足可能导致各种疾病。这种现象背后的潜在机制可能是，母亲的阳气不足，使胚胎在怀孕期间易受寒邪侵袭。中医理论认为，过多的体脂肪与阳气不足有关。肥胖母亲的卵子质量较差，因为肥胖可能损害卵母细胞和受精卵的发展。这表明阳气不足与卵子质量差之间可能存在相关性。

从线粒体的角度也可以尝试解释这些现象：报道称肥胖母亲的卵细胞存在线粒体缺陷，导致后代的正常线粒体数量少于对照组。研究发现，选择“好”的线粒体（mtDNA 突变很少且功能活性优良）保存并注入卵细胞中，而在卵发育过程中会清除“坏”的线粒体，保留“好”的线粒体。如果这些过程出现错误或不完整，可能导致新生儿疾病。最近的研究揭示了线粒体碎片化在女性生殖系中净化选择 mtDNA 的关键驱动因素，这一过程通过选择性去除有害的 mtDNA 来进行。

昼夜节律

根据中医理论，可以用阴阳理论来解释昼夜交替对人体生理状态的影响。《素问》中描述，阳气在白天占主导地位，从黎明开始出现，中午时分生长繁荣，到了夜晚则逐渐衰弱。当阳气旺盛时，人体呈现兴奋和活跃的状态，有助于人们在白天高效地学习和工作。到了晚上，阴气开始显现并繁荣，阳气隐退，这抑制并排斥了人体的兴奋和活跃状态，使人们进入睡眠。随着黎明的到来阳气再次出现并增长，人们逐渐醒来。因此，人体的生物钟（生物节律）实际上可能反映了阳气节律的运作。

类似于阳气的生物节律，线粒体与昼夜节律之间也观察到了关系。线粒体可能通过其对生物合成途径的影响或其在确定离子梯度中的作用，在昼夜节律机制中发挥作用。

事实上，线粒体具有自身的节律，包括形态（如形状和大小、融合和裂解）、呼吸作用、新线粒体的生成以及线粒体依赖的 ROS 产生，都与有效的昼夜钟相关。

全血 mtDNA 拷贝数显示出每日的波动节律，这是由于白细胞和血小板内容比例的波动。

此外，线粒体功能的波动也在全身能量消耗中发现，显示了人体骨骼肌氧化能力的昼夜节律。

免疫

中医理论认为，人体具有抵抗病理因素的能力，这主要得益于阳气。精神活动消耗了大量阳气，一旦阳气大量流失，就会引发疾病。

《灵枢》记载，大多数疾病在白天保持稳定，但在夜间会加剧，这与阳气耗竭状态有关。白天阳气占优势时，病理因素的数量和活性降低；夜间阳气不足时，病理因素可能变得猖獗和活跃，使疾病恶化。这表明疾病状态随着阳气总量的变化而变化，阳气与免疫功能密切相关。此外，线粒体也可以调节人体免疫。

线粒体与细胞凋亡/死亡密切相关，某些病毒蛋白和细菌效应物通过靶向线粒体来操纵细胞死亡和/或宿主细胞生存途径。线粒体是人体免疫的中心平台，线粒体信号传导对于启动针对各种入侵者的先天和适应性免疫反应是必需的。线粒体相关病毒传感器（MAVS）在抗病毒免疫中扮演着关键角色，连接线粒体功能和免疫信号。有趣的是，mtDNA 在免疫反应和炎症病理中扮演着至关重要的角色，它可以作为一种强大且高度潜在的炎症触发因子被释放，观察到在多种病理情况下的血细胞外部。

mtDNA的缺失会导致炎症体激活缺陷。值得注意的是，mtDNA与细菌基因组有许多共同特征（线粒体被认为是Alphaproteobacterium种的后代），因此细胞外mtDNA有潜力引发免疫反应并产生促炎信号。一些研究发现，在压力条件下线粒体增加了mtDNA的分泌，长期的压力最终可以导致各种疾病。例如，与对照组相比，肉芽肿性多血管炎患者的mtDNA水平显著升高。电针可以改善阳虚老年大鼠的抗疲劳能力，这可能与其降低肝线粒体的氧消耗和增加呼吸控制率有关。

经络

中医理论中描述了一种名为经络的低液阻通道系统，这是一种天生的体液循环系统，具有贯穿全身的能量网络特性，经络连接每一个器官并遍布全身表面，将体内外完美整合。《黄帝内经》指出，经络的功能是输送气血和循环阴阳以滋养身体，只要阳气在经络中自由顺畅地流动，各器官就能和谐工作，身体便保持健康；然而，一旦经络中的阳气受阻、中断或失衡，就会引起疾病症状。

阳气虚证相关疾病与线粒体

中医认为，人体保持健康状态需要阴阳动态平衡，否则会受到疾病的侵袭。《黄帝内经》记载，若体内元气旺盛，病理因素将受到抑制；实际上，作为一种元气的阳气，在维护人体健康中起着至关重要的作用，它是抵抗病理因素、预防和治疗疾病的主要力量。此外，线粒体与阳气在多方面有相似之处，例如，用草药补气的生物过程与线粒体途径密切相关，线粒体的生物功能与这些补气草药的基本药理效应一致，如调节凋亡、清除ROS、激活免疫系统、抗疲劳和提高运动能力。中医视心脏阳气不足为慢性心力衰竭的根本原因，而现代医学也发现线粒体功能障碍可导致心力衰竭。尝试从线粒体的角度解释由阳气衰减引发的疾病，包括mtDNA突变、ETC效率降低、ROS产生等问题。

癌症、慢性疲劳综合症（CFS）、睡眠障碍、老年性痴呆、代谢性疾病、

根据中医理论，阳气是维持人体生物活动的基本动力，阳气不足常伴随着疲劳综合症的高发，即使休息也难以缓解。中医将慢性疲劳综合症（CFS）描述为由阳气不足引发的疲劳综合症，并有多种传统草药用于改善和治疗 CFS 症状。西医着重于缓解 CFS 的症状，而中医通过草药、针灸和艾灸结合心理治疗提供全面持久的治疗。临床证据显示CFS 与线粒体功能障碍有关，各种线粒体功能异常可能是 CFS 症状的关键因素，并部分解释了 CFS 中观察到的代谢异常。

此外，睡眠被视为阴阳在体内平衡和相互作用的结果，失眠是“白天低精神状态”和“夜间过度兴奋状态”的恶性循环结果。阴阳失衡会导致失眠，通常是因为夜间阴气不足而阳气过盛。针灸常用于治疗失眠，有助于调节和恢复阴阳的自然节律。研究表明，线粒体与睡眠障碍相关，线粒体疾病的患者特别容易受到睡眠剥夺的影响，因为他们在氧气消耗和能量生产上存在困难。

根据中医理论，随着年龄增长，特别是超过50岁后，由于阳气不足导致的身体机能下降可能会引起一系列不利症状，如言语障碍、视听不稳和健忘，这与现代医学描述的老年痴呆症状一致。阳气长时间不足会导致其基本功能无法正常执行，可能导致痰和血在大脑中积聚和滞留，最终引发老年痴呆。

现代医学研究显示，线粒体功能障碍在老年痴呆中起着关键作用，发现线粒体的多种异常，包括mtDNA 突变、线粒体酶功能缺陷、电子传递链（ETC）功能减弱、线粒体形态改变、自噬、裂解-融合动力学和生物生成等问题，这些都与代谢和能量缺乏有关。同时，肥胖在当今社会普遍存在，中医认为九成肥胖人群存在阳气不足，阳气作为生命的基础，具有温暖全身和维持器官正常功能的作用，阳气不足会导致人体生理功能降低和新陈代谢减缓，从而导致体内脂肪代谢减慢和脂肪积累，进而引起肥胖。此外，现代医学研究显示，线粒体在脂肪代谢中起核心作用，线粒体功能异常导致脂肪酸氧化缺陷和脂质积累，与肥胖和2型糖尿病的发生有关。因此，从线粒体功能障碍解释阳气相关的代谢性疾病是有科学依据的。

结论与展望

阳气理论是《黄帝内经》中的一个重要学术思想，长期以来在中医中被高度重视。

遗憾的是，据我们所知，目前还没有直接有效的方法来准确测量阳气。这个测量问题是阻碍中医融入西医和科学研究的关键问题之一。中医通过望、闻、问、切四诊方法来判断一个人是否缺乏阳气，但这在现代医学中难以实施，成为整合东西方科学方法的主要障碍。从传统角度看，阳气代表“物质能量”或“能量流”，通过经络运行并遍布全身以维持人类生命活动，是人类生长、发展和繁殖的根本。巧合的是，线粒体也代表能量并与生命密切相关，没有线粒体，人类无法生存。

本文提出一种新视角，即线粒体可能是人体阳气的本质。

  1. 首先，线粒体和阳气在多个方面显示出相似性，我们对其中六个方面进行了详细分析，包括调节和维持体温、随年龄变化、新生儿疾病、昼夜节律、免疫反应和流通渠道。
  2. 其次，我们尝试找出与阳气不足相关疾病（如癌症、慢性疲劳综合症、睡眠障碍、老年痴呆和代谢病）与线粒体的关联，并从线粒体的角度解释这些疾病。
  3. 此外，还有一些临床上常见的阳气不足疾病，如甲状腺功能减退、性功能下降、夜尿增多和水肿。然而，由于目前研究量有限，尚不清楚线粒体功能障碍是否可以用来解释这些疾病的原因和发展。我们的分析将中医与现代线粒体医学联系起来，可以帮助在现代医学中确立中医阳气的重要功能视角。
  4. 此外，中医中的一些概念也可以用来指导线粒体研究。大量实验证据表明，许多增加阳气的中草药也可以增强线粒体功能。反之，我们不知道是否能增强线粒体活性的药物（如烟酰胺核苷酸、烟酰胺单核苷酸和辅酶 Q10）也能增加阳气。

中医认为一类能改善阳气的食物可以帮助治疗由阳气不足引起的疾病，但它们是否也可以贡献于现代医学中与线粒体相关疾病的治疗？发现更多支持线粒体作为人体阳气本质的证据将是一件有趣的事情。
