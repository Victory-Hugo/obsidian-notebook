摘要

Nuclear-embedded mitochondrial DNA sequences in 66,083 human genomes
https://www.nature.com/articles/s41586-022-05288-7

评论：66,083个人类基因组中的核嵌入线粒体DNA序列

DNA从细胞质细胞器转移到细胞核是内共生事件的遗留物——大多数核-线粒体片段（NUMT）被认为是古老的，早于人类物种形成。

作者分析了来自66,083人（包括12,509名癌症患者）的全基因组序列，并证明了线粒体DNA持续转移到细胞核中，从而形成了复杂的NUMT景观。

超过99%的个体至少具有1，637种不同NUMT中的一种，其中1/8的个体具有在不到0.1%的人群中存在的超罕见NUMT。我们评估的现存NUMT中有90%以上是在人类与类人猿分化后插入核基因组的。

一旦嵌入，这些序列就不再受到线粒体内的进化约束，并且NUMT特异性突变与线粒体DNA具有不同的突变特征。在生殖系中，每10^4例新生儿中观察到一次从头NUMTs（De novo NUMTs），每10^3例癌症中观察到一次从头NUMTs（De novo NUMTs）。

NUMTs优先涉及非编码线粒体DNA，将转录和复制连接到其起点；核插入则涉及多种机制，包括与PR结构域锌指蛋白9（PRDM9）相关的双链断裂修复。

肿瘤特异性 NUMT 的频率因癌症而异，包括粘液样脂肪肉瘤（myxoid liposarcoma）中的可能因果插入。我们发现了基于大小和基因组位置的针对NUMT的选择证据，形成了高度异质和动态的人类NUMT景观。

背景

从细胞质细胞器到细胞核的基因转移支持了线粒体起源的内共生理论。高等生物的线粒体基因组逐渐变小，反映了线粒体基因在进化过程中向核基因组的易位，促进了细胞器蛋白质通过胞质翻译机制的协调。许多NUMT在不同物种之间共享，反映了它们的古老起源。
最近的全基因组测序（Whole Genome Sequencing，WGS）发现许多超罕见NUMTs，证明NUMTs的产生是一个持续的过程。
新的mtDNA侵入核具有重要意义，因为它们可能破坏蛋白质编码基因，引起疾病，并产生类似于mtDNA混合群体的伪影（伪异质性(pseudo-heteroplasmy)）。
不经意将NUMT序列解释为mtDNA变异可能会混淆线粒体疾病的诊断，并提出线粒体DNA可能的父系遗传问题。在这里，描述了66083个个体的NUMT，包括8201个母亲-父亲-孩子三人组和12509个肿瘤-正常组织对，这些都是英国100000基因组计划中的。

人种系NUMT图谱

基于在53，535个个体（>99.9%）中检测到的至少两个不一致读段对，鉴定了335，891个参考序列中不存在的NUMT，包括3，829个不同的NUMT。
该更高严格性的数据集形成了本文报道的结果的基础，其中将NUMT称为

  1. 常见（频率（F）≥ 1%）
  2. 罕见（0.1% ≤ F < 1%）
  3. 超罕见（F < 0.1%）
  4. 私有（仅在一个家族中检测到）
    个体平均有4.7个在参考序列中不存在的NUMT，男女之间或随年龄增长无差异。大多数NUMT是短插入，NUMT大小与群体频率呈负相关。不同种族之间NUMT的频率和分布存在重大差异，非洲和东亚个体在NUMT频率和染色体位置方面最为明显。
    对3,197个三元组中发现的5,885个mtDNA-mtDNA分裂读段的分析表明，544个来自父亲，560个来自母亲。

所检查的NUMT显示相对于真正的mtDNA读数增加的CpG甲基化，真正的mtDNA读数没有甲基化，包括父代传递的串联的“mega-NUMT”，其在两代之间共享相同的甲基化模式，抑制它们在核基因组内的表达。串联NUMT可以模拟线粒体DNA的父系传递，产生类似线粒体DNA异质性的混合单倍型。

在每个 de novo NUMT 突变的情况下，研究还发现该 NUMT 的序列在孩子的核基因组中没有与其他地方对应，因此不太可能源自核DNA内部。这意味着这些 NUMT 序列不是从核基因组中其他地方插入的。
此外，研究还发现，在每个孩子及其父母中检测到的其他 NUMT 不与 de novo NUMT 插入的序列相匹配，即使通过降低从至少五个不一致的读数到两个不一致的读数的映射敏感性要求，仍然不相匹配。这表明 de novo NUMT 与其他 NUMT 是不同的序列。

NUMT插入的特征

研究了NUMTs的mtDNA和核DNA背景，NUMTs在所有核染色体上都有发现，并涉及整个mtDNA。3，184个相应的mtDNA断裂点在非编码置换环（D环）中富集——特别是在三个高变区以及重链和轻链起源；并且不太可能涉及[MT-ATP 6、MT-ND 2和MT-ND 3](线粒体DNA结构（mt-DNA structure）。
总的来说，观察到在考虑其他基因组特征后，每条染色体的长度与每条染色体上检测到的NUMT数量之间存在很强的正相关性。然而，染色体3、6和21具有比其余常染色体更大的NUMT数。X染色体每Mb的NUMT数量更少。在X染色体上观察到228个NUMT，女性的预期数量大约是男性的两倍。由于复杂的重复结构限制了置信比对，未分析Y染色体。

肿瘤特异性NUMT图谱

总体而言，肿瘤的平均NUMT数量较高。这种差异可能反映了肿瘤本身，而不是正常组织，因为不同正常组织类型之间的平均NUMT数没有差异。对于大多数肿瘤，个体诊断时的年龄和NUMT的数量之间没有相关性。然而，老年人血液系统恶性肿瘤的NUMT平均数较低，可能反映了其起源于克隆性造血。
肿瘤特异性NUMT的平均数量与年龄不相关，这意味着它们在癌变过程中出现，而不是在癌症形成之前的整个生命过程中的体细胞中出现。 NUMT 分布的不同是由于局部序列特征、核基因组的不稳定性以及相对较少的自然选择机会这几个因素的共同影响。这些因素使得 NUMT 在核基因组中形成了一种不同于生殖细胞中的分布格局。

NUMT插入的不良后果

在基因区域中观察到946个种系NUMT，其中大多数（85.8%，n = 812）富集在内含子中而不是外显子中。在编码DNA序列（CDS）中未发现常见或罕见NUMT，没有一个被预测会导致罕见疾病。
一个粘液样脂肪肉瘤肿瘤携带FUS-DDIT 3嵌合融合癌蛋白，该融合癌蛋白由涉及NUMT插入的复杂重排引起。在研究中，90%的粘液样脂肪肉瘤中存在FUS-DDIT 3融合蛋白，提示NUMT与个体的癌变有关。来自患有血液癌症的个体的两个正常组织携带极高数量的NUMT，其不存在于肿瘤组织中，可能反映克隆增殖。

NUMT序列的分子进化

我们估计了429例NUMT插入的年龄。其中绝大多数年龄小于10万年，41个年龄大于10万年，年龄范围可达375万年。正如预期的那样，年龄较大的NUMT在人群中更常见，特别是在非洲基因组中。并且比年轻的NUMT更容易携带NUMT特异性突变。

关于这一点，之后的研究进行了补充，可以参考这篇文献：Nuclear genetic control of mtDNA copy number and heteroplasmy in humans.

总之，这些发现表明在整个人类进化过程中NUMT的插入和进化正在进行。

NUMTs可以涉及整个线粒体DNA分子，但NUMT的断裂位点更常见于非编码D-环区域，包括重链和轻链复制的起始点。D-环区域是线粒体DNA的一个部分，其中包含了线粒体复制的起点。

最近描述的线粒体疝和BSX-BAK巨孔提供了一种途径，可能涉及RNA中间体泄漏到胞浆中，随后发生线粒体DNA的双链断裂（DNA双链断裂(DSB)）。

这段提到了最近关于线粒体疝和BSX-BAK巨孔的描述，这可能是NUMT形成的一个途径。这个过程涉及RNA中间体泄漏到细胞质中，然后线粒体DNA发生双链断裂（DSB）。
这也可能发生在单个细胞中，从而导致体细胞的镶嵌现象。

最后，这句话提出，这些过程也可能在单个细胞中发生，从而导致体细胞的镶嵌现象。这意味着不同细胞中的线粒体DNA拷贝数可能不同，形成了遗传上的多样性。
