摘要

研究表明，线粒体 DNA（mtDNA）突变的不同细胞内比例（异质性）与 mtDNA 疾病的表型异质性相关。常见病因 mtDNA tRNALeu(UUR) 3243A>G 突变的个体：

  1. 如果含有约10-30%的3243G 突变 mtDNA，表现为糖尿病，偶尔表现为自闭症；
  2. 含有约50-90%的突变 mtDNA 的个体则表现为脑肌病；
  3. 而含有约90-100%突变 mtDNA 的个体则面临围产期致死。
    为了确定这些突然的表型变化的基础，研究者创建了含有不同水平3243G 突变的体细胞融合体，并通过 RNA 测序分析了相关的细胞表型、核 DNA（nDNA）及 mtDNA 的转录特征。发现轻微增加突变 mtDNA 会导致氧化能力的轻微缺陷，但会引起细胞表型和基因表达的急剧转变。
  4. 含有20-30% 3243G mtDNA 的融合体显示 mtDNA mRNA 水平下降，线粒体形态圆润，细胞尺寸缩小；
  5. 含有50-90% 3243G mtDNA 的融合体表现出糖酵解基因的诱导、线粒体伸长、增加的 mtDNA mRNA 水平，以及信号转导、表观遗传调控和神经退行性疾病相关基因表达的变化。
  6. 最后，含有100% 3243G mtDNA 的融合体经历了 mtDNA 转录下降、线粒体形态圆润及核基因表达的变化。
    因此，响应 mtDNA 3243G 突变水平的轻微变化，nDNA 和 mtDNA 基因表达出现显著的阶段性变化。这表明，异质性 mtDNA 突变的表型变异的一个主要因素是核在响应线粒体逆向信号变化时能获得的状态数量有限。

重点

线粒体通过逆行信号（retrograde signaling）产生调节核基因表达的信号，但由于患者细胞内可能存在的突变体和正常线粒体 DNA 百分比的数量差异，使这种现象变得更加复杂。这项研究表明，根据其相对细胞质水平，单个 mtDNA 点突变可以在相同核背景的细胞内引起一组离散的细胞转录反应。这种通过 mtDNA 突变水平的定量变化对核基因表达的定性调节挑战了传统的“单一突变-单一疾病”概念，并为复杂代谢和退行性疾病、癌症和衰老的分子基础提供了另一种视角。

介绍

在人类16.6千碱基对的线粒体 DNA（mtDNA）中的突变可能导致一系列多系统疾病。与每个细胞仅含有两份的染色体基因不同，mtDNA 可以在一个细胞中存在数百到数千份。当细胞获得有害的 mtDNA 突变时，就会形成突变和正常 mtDNA 的细胞内混合体，这种状态被称为“异质性（Heteroplasmy）”。令人意外的是，异质性水平的相对微妙变化可能会对患者的表型产生重大影响，这一现象的细胞基础尚不明确。

这种效应最引人注目的例子之一是在核苷酸 3243A>G 处的 mtDNA 转移 RNA (tRNA)Leu(UUR) 突变中观察到的，这是最常见的人类致病性 mtDNA 点突变。

3243A>G 突变扰乱线粒体蛋白质合成，导致氨基酸错误掺入和电子传递链缺陷。当此突变的 mtDNA 异质性在50–90%时，可能引起多系统疾病，最常见的表型为线粒体脑肌病、乳酸血症和类卒中发作（MELAS）综合征。而较低异质性水平的同一突变与自闭症以及欧亚人群中的1型和2型糖尿病相关联，在极高水平时可表现为莱氏综合征或围产期致死。
mtDNA 3243A>G 异质性还会根据细胞系谱引发细胞类型特异的生化效应。考虑到不同细胞类型主要在表观遗传背景上有所不同，一种假设是连续的线粒体异质性变化可能导致离散的表型变化，因为细胞质的信号传导系统和核表观基因组可能只能以有限的几种离散方式响应线粒体功能的变化。这将导致基因表达轮廓的不连续变化，从而形成离散的临床表型[1]。

为了验证这一假设，研究人员制造了具有相同核背景但含有逐渐增加的3243A>G mtDNA 突变水平的线粒体细胞质杂交细胞（cybrids）。接着，比较了不同的 cybrid 系的细胞表型、线粒体功能和转录组状态。这种比较揭示了核 DNA 和 mtDNA 转录组对逐步增加的 mtDNA 异质性水平的显著非线性反应，伴随着转录轮廓的相似变化，这可能源于逆向信号对信号传导和表观遗传途径的影响。这种 mtDNA 异质性与基因表达轮廓之间的明显断裂可以解释同一 mtDNA 突变不同水平的患者之间显著的临床表型差异。

结果

本研究涉及从患有3243A>G 线粒体 DNA (mtDNA) 异质性的患者淋巴母细胞系中转移野生型（3243A）和突变型（3243G）mtDNA 到骨肉瘤143B 细胞（缺少胸苷激酶和 mtDNA）。

mtDNA 异质性改变线粒体功能、结构和细胞大小

  1. mtDNA异质性的建立与分析：

    • 将野生型和突变型mtDNA导入143B细胞，通过mtDNA的消耗、再增殖和克隆，成功建立了含有0%至100% 3243G突变mtDNA的稳定杂交体细胞系列。

  2. 电子传递链蛋白亚基的表达影响：

    • 随着3243G 突变水平的增加，mtDNA 编码的电子传递链蛋白亚基的表达呈剂量依赖性减少，但在突变率超过60%后，线粒体的最大呼吸能力仍然得以保持，直到90%的突变率时降低至接近ρo 细胞水平。

  3. 线粒体形态与功能的变化：

    • 线粒体形态与生物能量学密切相关，3243G突变型线粒体在突变率0%至90%逐渐伸长，在100%时恢复至接近正常水平。相比之下，ρo细胞的线粒体呈现极度碎片化和肿胀。

  4. 线粒体超微结构与细胞大小的变化：

    • 增加的mtDNA异质性导致线粒体结构、基粒和基质的电子密度以及细胞质中核糖体含量发生变化。
    • 与仅含有野生型mtDNA的细胞相比，含有20%和30% 3243G突变mtDNA的细胞显示出显著的横截面积和体积减少。细胞大小随后在50%突变率时增加，60%时减少，而在90%-100%突变率和ρo细胞中恢复至80%-90%的正常体积。

  5. 核-细胞比例与mtDNA复制数的动态变化：

    • 每个异质体水平的细胞mtDNA复制数也显示出相应的变化，尤其是在20%-30%的3243G水平时显著下降，50%时恢复正常，60%时再次显著下降，90%-100%时恢复正常。

对线粒体转录组的影响

  1. mtDNA基因编码及其表达：

    • mtDNA编码37个基因，包括两个rRNA，22个tRNA，以及13个关键的氧化磷酸化（OXPHOS）多肽mRNA基因，涉及复合体I至V的组成。
    • mtDNA的转录通过位于控制区的三个启动子进行，涉及一个富含C的轻链和两个富含G的重链，所有mtDNA链均以多基因簇连续转录方式表达，成熟的rRNAs和mRNAs通过切割tRNA释放。

  2. 突变对mtDNA转录的影响：

    • 在仅含有野生型3243A mtDNA的细胞中，mtDNA转录物占总细胞转录组的23.0%。但在含有20% 3243G突变mtDNA的细胞中，这一比例降至9.7%，30%突变时为12.7%，与细胞体积减小的比例相似。
    • 在含有60%至90% 3243G突变mtDNA的细胞中，mtDNA转录水平恢复至接近正常水平，而在100%突变的细胞中再次下降。

  3. 特定表达模式的变化：

    • 在所有异质性细胞中，除了60% 3243G的细胞线，mtDNA mRNA基因的表达普遍减少。
    • 突变tRNALeu(UUR)基因在20%至30%异质性细胞中被选择性诱导表达，增加5至8倍，而其他mtDNA tRNA基因的表达与其在mtDNA中的位置部分相关。

  4. 转录和tRNA基因表达的调节：

    • 在50%至60%的3243G突变负荷中，mtDNA RNA聚合酶（POLRMT）显著上调，然后在90%突变负荷时部分下降，100%时恢复至基线水平。
    • tRNA的表达在较低的突变负荷时上调，在较高的突变负荷时下调，表明在30%至50%异质性之间可能存在功能性转换。

功能阈值在 30% 到 50% 3243A>G 异质性之间诱导出不同的信号转导系统

  1. 糖酵解基因表达的调节：

    • 在20%至30%的3243G突变mtDNA中，糖酵解基因的表达水平与仅含有野生型mtDNA的细胞相似。这表明在这一突变水平下，并没有补偿性地上调糖酵解基因来弥补减少的mtDNA基因表达和细胞能量产生能力的下降。
    • 当3243G突变水平达到50%时，糖酵解基因的表达开始增加，并在90%突变时达到最高，显示出糖酵解酶基因的协同上调。

  2. 能量代谢的转换：

    • 随着糖酵解基因表达的上升，与细胞体积的恢复相对应，这表明细胞核在30%至50%的3243G突变间跨越了一个能量阈值，从主要的氧化代谢转向糖酵解代谢。
    • 当3243G突变水平达到100%时，糖酵解基因表达下降，并在ρo细胞中保持低水平，这些细胞缺乏功能性的呼吸链。

  3. 线粒体内膜ADP/ATP转运蛋白(ANT)的表达变化：

    • ANT1、ANT2和ANT3是线粒体内ADP/ATP转运的主要形式。ANT1的mRNA在20%至30%的3243G突变中上调，然后在50%至100%的3243G突变中下调，这与细胞能量需求高的组织（如肌肉、心脏和大脑）中ANT1的表达偏好一致。
    • ANT2在20%至30%的突变中表达增加，随后下降，并在90%的3243G突变中再次达到高峰，这与其在糖酵解代谢中的作用（将糖酵解生成的ATP导入线粒体以支持在OXPHOS ATP产生低下时的线粒体功能）相符。
    • ANT3的表达从0%开始一直上升至90%的3243G突变，然后在100%的突变中下降，其表达的显著诱导是前所未有的。

增加 mtDNA 异质性诱导转录谱中的相变

  1. 转录网络的阶段性变化：

    • 相对于0%的3243G控制组，20-30%、50-90% 和100% 3243G的mtDNA杂交体在功能通路的富集上显示出显著的不连续性。

  2. 关键通路的调节：

    • 在20-30% 3243G的杂交体中，PI3激酶-AKT、proto-oncogene myc和哺乳动物雷帕霉素靶蛋白（mTOR）通路被下调，而表皮生长因子（EGF）、激活转录因子5（ATF5）和骨形态发生蛋白2（BMP2）的基因靶点被上调。
    • 在50-90% 3243G的杂交体中，FOXP3、转录因子21（TCF21）和信号转导和转录激活因子3（STAT3）通路被下调，而缺氧诱导因子1（HIF1）、myc、K-Ras、PGC1α以及“衰老”和“衰老前期”通路被上调。
    • 在100% 3243G的杂交体中，干扰素β（Ifn-β）、多梳抑制复合体2（PRC2）、STAT3、指定蛋白1（SP1）和ETS相关基因1（EGR1）通路被下调。

  3. 转录表达的阶段性区分：

    • 分析表明，211个基因在20-30%的杂交体中被诱导表达，但在60-100% 3243G的杂交体中被超过100倍抑制。这些基因主要包括跨膜蛋白和信号传导元件，包括G蛋白偶联受体系统。
    • 主成分分析（PCA）进一步突显了0%，20-30%，50-90%，和100% 3243G及ρo细胞之间的转录表达配置文件的显著区别。

  4. 功能通路的独特性：

    • 与50-90% 3243G的差异性基因表达配置文件相关的分析显示，这些杂交体显著上调了线粒体功能（OXPHOS，TCA等）基因，与高突变水平时PGC-1α的上调一致，并与核糖体生物生成和端粒调控相关。
    • 50-90% 3243G的杂交体的基因表达配置文件与阿尔茨海默病、帕金森病和亨廷顿病等神经退行性疾病报告中的配置文件极为相似。

讨论

此研究展示了mtDNA异质性的连续变化如何通过信号传导和表观遗传调控过程的改变，导致核DNA和mtDNA基因表达配置文件的不连续重塑。这一现象挑战了传统医学视角中基于孟德尔遗传学规则预测基因突变所致表型的观点。研究指出，单一的mtDNA异质性核苷酸替换百分比的变化可以引起多种细胞表型，这通过类似阶段的基因表达配置文件的改变来实现，对年龄相关疾病和衰老的生化与分子基础具有深远的影响[2]。

  1. mtDNA和核DNA表达的调控：

    • mtDNA 异质性水平的变化会影响细胞能量代谢途径的激活，这种变化在不同异质性水平（20-30%、50-90%、100%）的杂交体细胞中表现尤为明显[3] [4]。

  2. 疾病表型与mtDNA异质性：

    • 不同的mtDNA异质性水平与不同的疾病表型（如糖尿病、自闭症、神经肌肉疾病）之间存在明显的关联，这表明mtDNA异质性可能是临床上观察到的mtDNA疾病表型变异的一个解释。

  3. 能量代谢的调节：

    • 在20-30% 3243G 异质性的糖尿病范围内，由于线粒体功能显著下降而未见到糖酵解的补偿性上调，这可能解释了 II 型糖尿病与线粒体生物能力缺陷的关联[5]。
    • 在50-90% 3243G水平，糖酵解被诱导以部分补偿OXPHOS缺陷，这与Alzheimer和Parkinson等神经退行性疾病的基因表达配置文件相关[6]。

  4. 线粒体和核基因表达的互动：

    • 本研究还显示，线粒体功能的改变可以影响染色质重塑，可能是通过线粒体产生的代谢中间体实现的，这反映在DNA甲基化机器基因的表达变化上。

  5. 实验模型的适用性和局限性：

    • 虽然使用的是源自骨肉瘤的细胞模型，可能与分化组织的生物能量动态不同，但实验结果对于非癌细胞中可能发生的基因表达变化具有参考价值。
    • 此外，实验中还考察了不同的核环境（如SH-SY5Y神经母细胞瘤核环境）对于3243G异质性mtDNA的影响，以排除核基因背景对观察到的转录阶段变化的影响。
