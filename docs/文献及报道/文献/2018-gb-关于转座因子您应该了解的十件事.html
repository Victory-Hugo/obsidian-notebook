摘要

Ten things you should know about transposable elements

基因组学的进展以及大规模功能检测的发展，表明转座子的活动在多方面对基因组及生物体生物学产生了深远影响。转座子不仅不应被忽视，其与细胞环境复杂的相互作用对理解基因组功能及生物学后果至关重要。本文以哺乳动物为例，详细阐述了转座子(Transposable elements, TEs) 的有趣事实。

介绍

转座子（TEs）是能在基因组中改变位置的DNA序列，具有极大的形态多样性。它们因其悠久的进化历史和持续的多样化而出现多种形式。

分类

转座子（TEs）根据其转座机制可以分为两大类，每一类又可以根据其在染色体上的整合机制进一步细分为不同的亚类。

  • 第一类（I类）转座子：称为逆转座子（retrotransposons），通过“复制-粘贴”的机制移动，即RNA中间体被逆转录为cDNA，并插入基因组的其他位置。这类包括：
    • LTR逆转座子：通过类似逆病毒整合酶催化的切割和链转移反应整合到染色体中。
    • 非LTR逆转座子（如 长散布核元件(Long Interspersed Nuclear Elements, LINEs)和短散布核元件(Short Interspersed Nuclear Elements, SINEs)）：通过靶引导逆转录整合。
  • 第二类（II类）转座子：称为DNA转座子，直接通过DNA中间体移动。这类包括：
    - “剪切-粘贴”机制：直接切割并插入新位置。
    - Helitrons转座子：通过“剥离-粘贴”机制复制，涉及环状DNA中间体。
    每个TE亚类进一步分为不同的超家族，它们在不同的生物中广泛分布，具有共同的遗传结构和单系起源。

转座子在基因组中的分布并非随机

转座子（TEs）在基因组中的分布并不随机，它们在特定的基因组区域或结构中有插入偏好。这种分布由转座子的自我复制需求与避免对宿主细胞功能产生有害影响的平衡力量所驱动。例如，在多种生物中，某些逆转座子倾向于插入RNA聚合酶III转录的基因上游区域，既不影响宿主基因表达，又能确保自身被转录。
自然选择和遗传漂变也是塑造转座子分布的重要力量。对宿主有强烈不利影响的转座子会被快速移除。对宿主影响不大的插入可能在不同物种中以不同效率达到固定化。
人类LINE 1逆转座子倾向于插入并破坏基因外显子，但很少有L1插入固定在基因编码区。LTR逆转座子在插入时不偏向DNA链的转录方向，但其在与转录方向一致的内含子区域中显著减少，可能因为会干扰基因剪接和多聚腺苷化。尽管不同哺乳动物物种的转座子内容存在差异，但转座子积累的进化轨迹在物种间具有一定的保守性。

转座子是突变和遗传多态性的主要来源

转座子占据了物种基因组的很大一部分，尤其是那些独特于该物种的DNA区域。例如，在玉米中，60-70%的基因组是由LTR逆转座子组成的。尽管DNA转座子在数量上较少，但在许多物种中它们是目前最活跃且具有致突变性的转座子类型。
在实验室中分离的果蝇表型突变体中，超过一半是由转座子自发插入引起的。在实验小鼠中，LTR元件的持续活跃决定了10-15%的遗传性突变表型。在人类基因组中，约有50万份L1逆转座子拷贝，但99.9%以上的L1拷贝已被固定，且由于突变和截短，无法再移动。每个人大约携带100个活跃的L1拷贝。转座子在有压力的环境下可能更加活跃。尤其是在自然环境中，转座子对遗传多样性的贡献可能被低估。大多数转座子插入对宿主没有直接的适应性优势，它们主要通过遗传漂变达到种群固定，并随着时间推移中性积累的点突变会使这些转座子失去转座功能。
水平转座子转移（跨个体或跨物种的转座子传播）是一种常见现象，影响几乎所有类型的转座子和生物系统发育树上的所有生物[1] [2]。尽管水平转移的细胞机制尚不清楚，但转座子的内在流动性和宿主物种间的生态相互作用（如病原体和寄生物）促进了跨物种的转座子传播。人类“参考基因组”并未包括所有转座子，个体之间的单倍体基因组在约1000个转座子插入位点上存在差异，主要来自L1或Alu家族。

转座子与基因组重排和独特染色体特征有关

转座活动是基因组扩展的强大机制，但随着时间推移，DNA的删除机制平衡了这一过程，驱动了真核生物基因组大小的演化。由于转座子插入和删除往往不精确，它们可以间接影响周围的宿主序列，导致大量的基因和调控序列的重复和重排。转座子的转座速率部分受宿主控制，是基因组进化的重要推动力。
转座子失去移动能力后，仍可通过与分散在远距离位置的相关转座子发生重组，引发大规模的删除、重复和倒位。转座子提供的微同源区在复制修复中易于模板切换，导致结构变异的产生。
在果蝇中，LINE样逆转座子替代了端粒酶的功能，维持端粒，这一现象可以被视为对染色体“末端问题”的进化解决方案[3]。端粒酶的逆转录酶成分可能源自古老的逆转座子家族 [4] [5]。转座子序列和转座酶基因在着丝粒中也发挥结构作用。

转座子表达和抑制存在内在平衡

转座子（TEs）在进化过程中需要维持表达与抑制之间的平衡。适度的表达有助于扩增转座子数量，但过度表达可能对宿主造成不利影响，导致宿主适应性下降，从而不利于转座子的长期存续。宿主通过多种机制控制转座子表达，包括小RNA、染色质修饰、DNA修饰以及序列特异性的抑制因子。
在某些发育阶段，特别是在胚胎早期发育中，宿主必须暂时解除部分抑制机制，以允许宿主基因表达的调控，例如原始生殖细胞中的基因印记重置所需的全基因组DNA去甲基化，这同时可能激活转座子。在一些生物中，宿主采用巧妙机制防止转座子插入会被遗传。
转座子的表达和对宿主的影响在不同的组织类型和生物生命周期的不同阶段会有显著差异。理想情况下，转座子应在生殖细胞中表达，而不是在体细胞中，因为在体细胞中的表达对转座子没有遗传优势。例如，在纤毛虫中，转座子会从体细胞的巨核中删除，但保留在微核（生殖细胞）中。植物中生殖细胞通常在发育晚期由体细胞分化而来，因此转座子在植物的体细胞中发生的转座事件有可能被遗传下来，这与有明确生殖细胞系分化的动物存在区别。

转座子是生殖系和体细胞中的插入突变原

在人类中，目前仍有活跃的转座子（如L1及依赖于L1编码蛋白的移动元件），它们的表达与抑制之间仍然保持平衡。这些活跃转座子会导致生殖系插入突变，引发遗传疾病。在人类中，Alu元件的生殖系转座率约为每21次出生中发生1次，L1元件则为每95次出生中发生1次。
尽管体细胞转座传统上被认为对进化没有长期影响，但已有大量证据表明，转座子在许多生物的体细胞中也是活跃的。人类的L1表达与转座活动已在多种体细胞环境中被检测到，包括早期胚胎和某些干细胞。特别是在哺乳动物的大脑中，L1转座子被认为可能通过增加神经元细胞群的多样性来发挥作用。
在人类癌症中，也观察到了大量的体细胞L1插入，某些肿瘤可以获得数百个新的L1插入。少数“热点”L1位点在个体、肿瘤类型及肿瘤克隆进化的不同阶段具有活跃性。一些新的L1插入可能破坏了关键的抑癌基因或癌基因，推动癌症形成。宿主细胞进化出多种机制来控制转座子活性。然而，随着自然选择的力量随着年龄的增长逐渐减弱，特别是在生殖期之后，转座子的活性可能会增加。

转座子对宿主的损害不仅仅来自转座活动

转座子最为人知的危害是其移动性，但这并不是转座子对宿主的唯一或最常见的损害方式。

  • 转录失调：转座子的去抑制和过度转录会干扰宿主mRNA的转录和处理。
  • 基因组不稳定：转座子编码的蛋白质可能导致DNA断裂，进而引发基因组不稳定。
  • 免疫反应：转座子衍生的RNA和细胞外DNA拷贝可能触发先天免疫反应，引发自体免疫疾病和无菌性炎症。

参与逆转录过程的转座子编码蛋白及其产生的细胞质DNA和RNA杂合物可激活炎症通路。一些内源性逆转录病毒可编码功能性蛋白，与多发性硬化症和肌萎缩性侧索硬化症等神经退行性疾病相关。

许多关键的编码和非编码 RNA 均源自转座子

转座子插入可以为蛋白编码基因和非编码RNA的产生提供原始材料，某些转座子来源的RNA已经获得了重要的细胞功能，甚至是必不可少的功能。
转座子基因适应（Exaptation）过程在进化中发挥重要作用，产生了深度保守的功能和物种特异性特征。

转座子为顺式调控元件做出贡献并修饰转录网络

转座子（TEs）为调控真核基因表达提供了丰富的材料，包括启动子、增强子、转录因子结合位点、绝缘子序列以及抑制元件。这些元件可以影响基因表达网络，协调多个基因的转录以控制复杂的生物学过程。
由于转座子家族通常以大量相关拷贝填充基因组，长期以来推测它们可以向不同位置的基因提供相同的顺式调控模块，从而将分散的基因“连接”起来，协同调控它们的表达。
转座子为进化过程中顺式调控网络的构建和重塑提供了基础。研究表明，转座子参与了多种生物学过程的调控，如妊娠、干细胞多能性、新皮质发育、哺乳动物的先天免疫。转座子序列具备经典基因调控网络的所有必要特征：它们能够被多种转录因子结合，整合多个输入信号（激活/抑制），并且能够协调调控基因表达。因此，转座子通过创建新的顺式调控回路和微调现有网络，在调控生物过程方面发挥重要作用。

分析转座子需要专门的工具

由于转座子高度重复的特性，它们在基因组研究中常常被忽视。分析转座子面临多种挑战，需要使用专门的工具和技术。在实验设计和数据分析中，必须考虑转座子区域的重复性或唯一性。在某些情况下，可以接受或期望同时靶向多个转座子元素，甚至整个转座子家族。不同物种的转座子活性和历史差异会影响分析的复杂性。例如，在小鼠基因组中，转座子的活跃程度比在人类基因组中高，因此比对和分析小鼠基因组中的转座子更加具有挑战性。随着测序技术和生物信息学分析管道的进步（特别是测序读取长度的增加），早期研究中面临的许多障碍将逐步被克服。

展望

功能性转座子的定义争议：ENCODE项目曾主张80%基因组有生化活动，但这一说法引发了关于功能性DNA的广泛争论。批评者认为功能的进化定义被忽略了。转座子作为“垃圾DNA”的组成部分，处于同样的争议中：虽然许多转座子具备生化活动（如启动子或增强子），但这些活动是否足以被视为具有功能性仍存疑。
物种间的差异与研究挑战：不同物种之间转座子的巨大差异使得建立转座子的调控作用更加复杂。
研究转座子功能的难题：通过基因敲除实验可以在体外测试特定转座子位点的调控作用，但由于某些转座子在模式生物中不存在，这类研究受限。
更严谨的功能性定义：目前的共识是，转座子的生化活动（如转录因子结合或位于开放染色质区域）不应简单等同于宿主适应性中的功能性。只有在显示出进化选择压力或基因改变后导致表型缺陷的转座子才应被视为具有功能性。
基因组编辑的前景：随着基因组和表观基因组编辑技术的进步，尤其是在重复元素上的应用，未来有望系统评估转座子的功能意义。
