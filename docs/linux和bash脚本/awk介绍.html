cut命令

假如我们有这样一个文件AA.txt

name age sex ID 
张三 25 男 512351189874510014
李四 23 女 512121149874510014

首先可以查看这个文件，使用命令：

cat AA.txt
符号 意义
-f 提取第几列
-d 分隔符是什么

-f 列号： 提取第几列
-d 分隔符： 按照指定分隔符分割列

如果我们想要把sex给提取出来，那么我们可以使用

cut -f 3 AA.txt

如果我们想把以:分割的第1列和第3列提取出来，那么我们可以使用

cut -d ":" -f 1,3 AA.txt

cut命令的局限性：不能分割不定长度的空格。

awk命令

把文件逐行的读入，以空格为默认分隔符将每行切片，切开的部分再进行各种分析处理。

语法：awk ‘条件1{动作1}条件2{动作2}...’文件名

eg:[root@qianfeng01 ~]# df -h | awk '{print $1 "\t" $3}' 显示第一列和第三列

的 | 符号。这个符号在 Unix 和 Linux shell 中称为管道（pipe）。将一个命令的输出传递给另一个命令作为输入。

练习题

这里有一份文件，假如文件名字是information.txt。

Name,Gender,Age
Alice,Female,30
Bob,Male,25
Charlie,Male,35
Diana,Female,28
  1. 我想把Age提取出来，怎么办？
awk '{print $3}' information.txt
  1. 想把Name,Age提取出来，并放到一个名为information2.txt的文件里，怎么办怎么办？
awk -F',' '{print $1 "," $3}' information.txt > information2.txt

-F, 指定字段分隔符为逗号（,）;如果原文件是以空格或制表符分隔的，需要相应地调整 awk 命令中的字段分隔符设置。
对于空格分隔的文件：不需要，因为 awk 默认使用空格作为字段分隔符。
对于制表符分隔的文件：制表符可以用 \t 表示。

  1. 假如我想把Age放在第一列，Name放在第二列，并放到一个名为information2.txt的文件里，怎么办？
awk -F',' '{print $3 "," $1}' information.txt > information2.txt
  1. 假如我想把源文件是以制表符分隔的，有些名字比如Alice_Hugo有下划线分割，我只需要Alice，不需要Hugo，那么我应该如何提取它们？
awk -F'\t' '{split($1, a, "_"); print a[1] "\t" $3}' information.txt > information2.txt
  • -F'\t' 设置字段分隔符为制表符。
  • split($1, a, "_") 仍然用于将第一个字段（$1，即 Name 字段）按下划线 (_) 分割，并将结果存储在数组 a 中。
  • print a[1] "\t" $3 表示打印出处理过的 Name 字段和原始的 Age 字段，两者之间用制表符分隔。
  • information.txt 是您的输入文件。
  • > 用于将输出重定向到 information2.txt 文件。
